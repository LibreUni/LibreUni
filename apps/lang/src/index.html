lang.libreuni

Language tracks can now ship independently.

This section is intentionally outside Astro so it can evolve on its own stack while still living inside the LibreUni repository and deployment model.

Cross-site progress

Keep room for shared auth-free progress sync or signed progress handoff from the main LibreUni experience.

Independent frontend

This app only needs a build script that outputs a dist/ folder, so it can later switch to any framework without changing the repo-level deployment shape.

Clean handoff

The main site can link or redirect here by subdomain while Coolify deploys it from the same repository with APP=lang.
