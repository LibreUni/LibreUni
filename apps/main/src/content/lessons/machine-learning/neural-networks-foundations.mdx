# The Foundation of Deep Learning

Artificial Neural Networks (ANNs) represent a shift from explicit algorithmic logic to a model of "connectionism." Inspired by the architecture of the human brain, these networks consist of interconnected processing units that learn to recognize complex patterns through iteratively adjusting their internal parameters.

## The Atomic Unit: The Perceptron

The **Perceptron**, developed by Frank Rosenblatt, is the simplest form of a neural network. It mimics the behavior of a biological neuron: it receives electrical signals (inputs), weights them by importance, and "fires" if the total signal exceeds a certain threshold.

### Mathematical Definition
For a vector of inputs $x = [x_1, x_2, \dots, x_n]$, the neuron calculates a weighted sum $z$:
$$ z = \sum_{i=1}^{n} w_i x_i + b = w^T x + b $$
where $w$ represents the **Weights** and $b$ is the **Bias**.

The result $z$ is then passed through an **Activation Function** $\sigma(z)$ to produce the output $y$.

## The Role of Activation Functions

Without activation functions, a neural network is just a series of linear transformations. Because the composition of linear functions is itself linear ($f(g(x)) = ax+b$), a 100-layer "linear" network would be no more powerful than a single-layer linear regression.

Activation functions introduce **Non-Linearity**, allowing the network to approximate complex, curvy decision boundaries.

### Core Activation Functions
1. **Sigmoid**: $\sigma(z) = \frac{1}{1 + e^{-z}}$. Maps output to $(0, 1)$. Historically popular but rarely used in hidden layers today due to the "vanishing gradient" problem.
2. **tanh**: Maps to $(-1, 1)$. Zero-centered, which often makes optimization faster than sigmoid.
3. **ReLU (Rectified Linear Unit)**: $\sigma(z) = \max(0, z)$. The gold standard for deep learning. It's computationally cheap and its derivative doesn't vanish for positive inputs.
4. **Softmax**: Used in the final layer for multi-class classification. It turns raw scores into a probability distribution that sums to 1.

## Architecture: The Multi-Layer Perceptron (MLP)

A single perceptron cannot solve the **XOR Problem** (it can only draw straight lines). To solve complex problems, we arrange neurons into three types of layers:

1. **Input Layer**: One node per feature in your dataset.
2. **Hidden Layers**: Layers between input and output where the "learning" happens. The **Universal Approximation Theorem** states that a network with just one hidden layer can approximate any continuous function, provided it has enough neurons.
3. **Output Layer**: Produces the final result (e.g., a single value for regression, or $K$ probabilities for $K$-class classification).

## How Neural Networks Learn

Learning is the process of finding the values for $w$ and $b$ that minimize a **Loss Function** (like Mean Squared Error or Cross-Entropy).

### 1. The Forward Pass
Data enters the input layer and ripples through the network. Each neuron calculates its activation and passes it forward. At the end, we get a prediction $\hat{y}$.

### 2. The Loss Calculation
We compare $\hat{y}$ to the real answer $y$ using a loss function. This tells us "how wrong" the network currently is.

### 3. Backpropagation (The Engine)
Backpropagation is the application of the **Chain Rule** from calculus. We calculate how much each individual weight contributed to the final error. 
- We start at the end and work backward.
- We find the gradient: $\frac{\partial \text{Loss}}{\partial \text{Weight}}$.

### 4. Weights Update (Optimization)
We use an optimizer like **Stochastic Gradient Descent (SGD)** or **Adam** to nudge the weights in the direction that reduces the error:
$$ w_{new} = w_{old} - \text{learning\_rate} \cdot \text{gradient} $$

## Implementation from Scratch: A Single Neuron

Let's see the inner workings of a single neuron's training loop.

<CodeRunner client:load
  code={`import numpy as np

# Training data (OR Gate)
X = np.array([[0,0], [0,1], [1,0], [1,1]])
y = np.array([[0], [1], [1], [1]])

# Initialization
weights = np.random.randn(2, 1)
bias = 0
lr = 0.1

for epoch in range(100):
    # Forward Pass
    z = np.dot(X, weights) + bias
    a = 1 / (1 + np.exp(-z)) # Sigmoid
    
    # Loss (MSE)
    error = a - y
    
    # Backprop (Gradients)
    # Derivative of Sigmoid is a * (1 - a)
    dz = error * (a * (1 - a))
    dw = np.dot(X.T, dz)
    db = np.sum(dz)
    
    # Update
    weights -= lr * dw
    bias -= lr * db

print("Prediction for [1,1]:", 1 / (1 + np.exp(-(np.dot([1,1], weights) + bias))))`}

## Common Challenges in Deep Learning

### 1. The Vanishing Gradient
In very deep networks using Sigmoid/Tanh, as we propagate the error backward, the gradients are multiplied by values between 0 and 1. By the time the error reaches the first layers, the gradient is so small ($0.00001$) that the weights barely change. This is why **ReLU** and **Batch Normalization** are essential in modern architectures.

### 2. Overfitting (High Variance)
Deep networks have millions of parameters. They are prone to memorizing the training set. We combat this with:
- **Dropout**: Randomly "turning off" neurons during training to force the network to become redundant and robust.
- **L2 Regularization**: Penalizing large weights.
- **Early Stopping**: Stopping training once the validation error starts increasing.

## Modern Variations

While MLPs are the foundation, different data types require specialized architectures:
- **Convolutional Neural Networks (CNNs)**: For grid-like data (Images). They use "Filters" to capture spatial features.
- **Recurrent Neural Networks (RNNs)**: For sequential data (Text, Time-series). They have "Memory" of previous inputs.

## Conclusion

Neural networks are the mathematical realization of learning through experience. By stacking many simple units and applying the chain rule of calculus, we can build machines that learn to recognize faces, translate languages, and drive cars. Understanding the flow of information—from linear sum to non-linear activation, and from global error to individual weight adjustment—is the key to mastering the modern AI landscape.
