Graphs provide a formal framework for modeling relationships between discrete objects. A graph $G = (V, E)$ consists of a set of vertices $V$ and edges $E$.

### Fundamental Properties

- **Degree**: The number of edges connected to a vertex.
- **Handshaking Lemma**: $\sum_{v \in V} \text{deg}(v) = 2|E|$.

### Connectivity and Traversal

- **Connected**: A path exists between every pair of vertices.
- **Cycle**: A path that starts and ends at the same vertex.

<CodeRunner 
  client:load 
  code={`def find_path(graph, start, end, path=[]):
    path = path + [start]
    if start == end: return path
    for node in graph[start]:
        if node not in path:
            newpath = find_path(graph, node, end, path)
            if newpath: return newpath
    return None

print(f"Path from A to F: {find_path(graph, 'A', 'F')}")`} 
/>

### Planar Graphs and Euler's Formula

A graph is **planar** if it can be drawn without edges crossing. For a connected planar graph:
$$V - E + F = 2$$
where $F$ is the number of faces.

### Special Graphs: Trees and Bipartite Graphs

- **Tree**: A connected graph with no cycles. A tree with $n$ vertices always has $n-1$ edges.
- **Bipartite**: Vertices can be partitioned into two sets such that no edge exists within a set.

<CodeRunner 
  client:load 
  code={`def is_bipartite(graph):
    color = {}
    for node in graph:
        if node not in color:
            stack = [(node, 0)]
            color[node] = 0
            while stack:
                u, c = stack.pop()
                for v in graph[u]:
                    if v in color:
                        if color[v] == c: return False
                    else:
                        color[v] = 1 - c
                        stack.append((v, 1 - c))
    return True

print(f"Is graph bipartite? {is_bipartite(graph)}")`} 
/>

### Famous Theorems

- **Four Color Theorem**: Any planar graph can be colored with 4 colors.
- **Kuratowski's Theorem**: A graph is planar iff it doesn't contain $K_5$ or $K_{3,3}$.
