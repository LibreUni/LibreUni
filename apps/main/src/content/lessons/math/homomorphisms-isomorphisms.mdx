# Group Homomorphisms and Isomorphism Theorems

How do we compare two completely different groups? For instance, does the group of rotations of a square behave like the arithmetic of integers modulo 4? 

To answer such questions, we need structure-preserving maps between groups. In abstract algebra, these maps are called **homomorphisms**. When a homomorphism is a perfect, one-to-one correspondence, it is an **isomorphism**, meaning the two groups are structurally identical.

## The Motivating Problem: Modulo Arithmetic

Consider the group of integers under addition $(\mathbb{Z}, +)$ and the finite group $(\mathbb{Z}/4\mathbb{Z}, +)$. Let's define a map:
$$\phi: \mathbb{Z} \to \mathbb{Z}/4\mathbb{Z} \quad \text{by} \quad \phi(x) = x \pmod 4$$

Does this map respect the group operations? That is, if we add two integers first and then apply the map, do we get the same result as mapping them first and then adding them modulo 4?
Let's check with $x = 5$ and $y = 6$:
- Sum then map: $\phi(5 + 6) = \phi(11) = 11 \pmod 4 = 3$.
- Map then sum: $\phi(5) + \phi(6) \pmod 4 = (1 + 2) \pmod 4 = 3$.

They match! This map preserves the addition operation. But what elements map to the identity element $0$ in $\mathbb{Z}/4\mathbb{Z}$?

### Try It Yourself: Finding the Kernel

The set of all elements in the domain that map to the identity in the codomain is called the **kernel** of the map.

## Formalization: Homomorphisms and Isomorphisms

Let $(G, \cdot)$ and $(H, *)$ be groups. A function $\phi: G \to H$ is a **group homomorphism** if for all $x, y \in G$:
$$\phi(x \cdot y) = \phi(x) * \phi(y)$$

If $\phi$ is also a bijection (both injective and surjective), it is called a **group isomorphism**, and we write $G \cong H$.

### Kernel and Image

For any group homomorphism $\phi: G \to H$:
- The **kernel** of $\phi$ is the subset of $G$ that maps to the identity $e_H \in H$:
  $$\ker(\phi) = \{ x \in G \mid \phi(x) = e_H \}$$
- The **image** of $\phi$ is the subset of $H$ containing all outputs of $\phi$:
  $$\text{Im}(\phi) = \{ \phi(x) \mid x \in G \}$$

### Properties of Homomorphisms

For any homomorphism $\phi: G \to H$:
1. $\phi(e_G) = e_H$.
2. $\phi(x^{-1}) = \phi(x)^{-1}$ for all $x \in G$.
3. $\ker(\phi)$ is a **normal subgroup** of $G$ (denoted $\ker(\phi) \trianglelefteq G$).
4. $\text{Im}(\phi)$ is a subgroup of $H$.

---

## The First Isomorphism Theorem

The most important theorem relating homomorphisms, normal subgroups, and quotient groups is the **First Isomorphism Theorem**:

> **Theorem (First Isomorphism Theorem):** If $\phi: G \to H$ is a group homomorphism, then the quotient group $G/\ker(\phi)$ is isomorphic to the image of $\phi$.
> $$G/\ker(\phi) \cong \text{Im}(\phi)$$

### Proof of the Theorem

Let $K = \ker(\phi)$. Define a map $\psi: G/K \to \text{Im}(\phi)$ by:
$$\psi(gK) = \phi(g)$$

1. **Well-defined:** If $g_1 K = g_2 K$, then $g_2 = g_1 k$ for some $k \in K$. Thus:
   $$\phi(g_2) = \phi(g_1 k) = \phi(g_1) \phi(k) = \phi(g_1) e_H = \phi(g_1)$$
   So $\psi(g_1 K) = \psi(g_2 K)$.
2. **Homomorphism:** $\psi((aK)(bK)) = \psi((ab)K) = \phi(ab) = \phi(a)\phi(b) = \psi(aK)\psi(bK)$.
3. **Injective:** If $\psi(aK) = \psi(bK)$, then $\phi(a) = \phi(b) \implies \phi(b)^{-1}\phi(a) = e_H \implies \phi(b^{-1}a) = e_H \implies b^{-1}a \in K \implies aK = bK$.
4. **Surjective:** By definition, every element in $\text{Im}(\phi)$ is of the form $\phi(g) = \psi(gK)$.

Thus, $\psi$ is an isomorphism!

## Contrast: Homomorphism vs Arbitrary Map

To see the strength of the homomorphism property, let's contrast it with a map that is not a homomorphism.
Consider the map $\theta: \mathbb{Z} \to \mathbb{Z}$ given by:
$$\theta(x) = x + 1$$
If we check the homomorphism property under addition:
- $\theta(x + y) = x + y + 1$
- $\theta(x) + \theta(y) = (x + 1) + (y + 1) = x + y + 2$

Since $x + y + 1 \neq x + y + 2$, this map is **not a homomorphism**. It does not preserve the additive identity (since $\theta(0) = 1 \neq 0$).
