## Python Lab: Symbolic LaTeX Generation

In serious mathematical computing, we often use libraries like `sympy` to generate LaTeX code for complex expressions automatically.

<CodeRunner client:load code={`import sympy as sp

# Define symbols
x, y = sp.symbols('x y')

# Create a complex expression
expr = sp.sin(x)**2 + sp.exp(x * y) / (x + y)

# Compute the derivative with respect to x
diff_expr = sp.diff(expr, x)

print("Original Expression (LaTeX):")
print(sp.latex(expr))

print("\\nDerivative Case (LaTeX):")
print(sp.latex(diff_expr))

# Generate a matrix
A = sp.Matrix([[1, x], [y, 1]])
print("\\nMatrix A (LaTeX):")
print(sp.latex(A))
`} />
