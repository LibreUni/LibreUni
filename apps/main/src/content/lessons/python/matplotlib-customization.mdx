# Plotting for Publication

Scientists don't just produce plots; they produce evidence. A good plot should be clear, accessible, and aesthetically pleasing. In this lesson, we explore how to move beyond default settings.

## Global Styles with `plt.style`

Matplotlib comes with several pre-defined style sheets. You can view them all using `plt.style.available`.

<CodeRunner client:load 
  title="Switching Styles"
  code={`
import matplotlib.pyplot as plt
import numpy as np

# Use a clean, modern style
plt.style.use('ggplot')

x = np.random.randn(100)
y = np.random.randn(100)

fig, ax = plt.subplots()
ax.scatter(x, y, alpha=0.5, color='purple')
ax.set_title('Styled Scatter Plot')

plt.show()
`}
/>

## Working with Colormaps

Choosing the right colormap is critical. Avoid "jet" and other non-perceptually uniform maps. Use `viridis`, `magma`, or `inferno` for continuous data.

<CodeRunner client:load 
  title="Colormap Comparison"
  code={`
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 5, 100)
y = np.linspace(0, 5, 100)
X, Y = np.meshgrid(x, y)
Z = np.sin(X)**10 + np.cos(10 + Y*X) * np.cos(X)

fig, ax = plt.subplots()
im = ax.imshow(Z, origin='lower', extent=[0, 5, 0, 5], cmap='viridis')
fig.colorbar(im, label='Intensity')
ax.set_title('Perceptually Uniform Colormap')

plt.show()
`}
/>

## Annotations: Highlighting Key Data

Sometimes you need to point directly at a feature in your data.

<CodeRunner client:load 
  title="Using Annotations"
  code={`
import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots()

t = np.arange(0.0, 5.0, 0.01)
s = np.cos(2*np.pi*t)
ax.plot(t, s, lw=2)

# Point to the peak
ax.annotate('Local Maximum', xy=(2, 1), xytext=(3, 1.5),
            arrowprops=dict(facecolor='black', shrink=0.05))

ax.set_ylim(-2, 2)
plt.show()
`}
/>

## Complex Layouts with GridSpec

While `plt.subplots()` handles simple grids, `GridSpec` allows for plots that span multiple rows or columns.

<CodeRunner client:load 
  title="GridSpec for Complex Dashboards"
  code={`
import matplotlib.pyplot as plt

fig = plt.figure(figsize=(8, 6))
gs = fig.add_gridspec(3, 3)

ax1 = fig.add_subplot(gs[0, :])
ax1.set_title('Top row, full width')

ax2 = fig.add_subplot(gs[1:, :2])
ax2.set_title('Bottom left, big')

ax3 = fig.add_subplot(gs[1, 2])
ax3.set_title('Small 1')

ax4 = fig.add_subplot(gs[2, 2])
ax4.set_title('Small 2')

plt.tight_layout()
plt.show()
`}
/>
