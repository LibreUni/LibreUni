# Rings, Fields, and Integral Domains

While group theory deals with a single binary operation, rings and fields formalize structures with two operations: **Addition** and **Multiplication**.

## The Definition of a Ring

A **ring** $(R, +, \cdot)$ satisfies:
1. $(R, +)$ is an **Abelian Group**.
2. Multiplication is **Associative**.
3. **Distributive Laws**: $a(b+c) = ab+ac$.

## Integral Domains and Zero Divisors

An **Integral Domain** is a commutative ring where $ab = 0 \implies a=0 \text{ or } b=0$. If an element $a \neq 0$ satisfies $ab=0$ for some $b \neq 0$, $a$ is a **Zero Divisor**.

  code={`def find_zero_divisors(elements, mult_op, zero):
    divisors = []
    for a in elements:
        if a == zero: continue

            if b == zero: continue
            if mult_op(a, b) == zero:

## Fields

A **Field** is an integral domain where every non-zero element has a multiplicative inverse. This allows for division.

<CodeRunner 
  client:load 
  code={`def get_units(elements, mult_op, one, zero):
    units = []
    for a in elements:
        if a == zero: continue
        for b in elements:
            if mult_op(a, b) == one:
                units.append(a)
                break
    return units

# Z/5Z (a field) vs Z/4Z (not a field)
mult5 = lambda a, b: (a * b) % 5
mult4 = lambda a, b: (a * b) % 4
print(f"Units in Z/5Z: {get_units(range(5), mult5, 1, 0)}")
print(f"Units in Z/4Z: {get_units(range(4), mult4, 1, 0)}")`}

## Ideals and Quotient Rings

An **Ideal** $I$ of a ring $R$ is a subring such that for every $r \in R$ and $i \in I$, the products $ri$ and $ir$ are in $I$. 
- Formally: $(R, +)$ is a subgroup, and $\forall r \in R, \forall i \in I: r \cdot i \in I \text{ and } i \cdot r \in I$.

If $I$ is an ideal, the **Quotient Ring** $R/I$ is the set of cosets $\{r + I : r \in R\}$ under the operations:
- $(a + I) + (b + I) = (a + b) + I$
- $(a + I) \cdot (b + I) = (ab) + I$

## Ring Homomorphisms

A mapping $f: R \to S$ between two rings is a **homomorphism** if it preserves both operations:
1. $f(a + b) = f(a) + f(b)$
2. $f(ab) = f(a)f(b)$
3. $f(1_R) = 1_S$ (for rings with identity)

The **Kernel** of a homomorphism $\ker(f) = \{r \in R : f(r) = 0_S\}$ is always an ideal of $R$.

### Principal Ideal Domains (PID)
A ring where every ideal is generated by a single element. $\mathbb{Z}$ is a PID because every ideal is of the form $n\mathbb{Z}$.
