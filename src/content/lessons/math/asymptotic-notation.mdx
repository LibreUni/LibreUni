# Asymptotic Notation

Comparing the growth rate of functions is essential for classification: which algorithms scale, which recurrences are tractable, and how resource demands grow with input size. The Landau symbols provide a vocabulary for comparing functions as their argument tends to infinity.

## Formal Definitions

Let $f, g : \mathbb{N} \to \mathbb{R}^{\ge 0}$ be functions. The Landau symbols describe the asymptotic relation between $f$ and $g$ as $n \to \infty$. Each symbol defines a **set** of functions: $O(g)$ is the set of all functions that grow no faster than $g$.

<MathStatement id="big-o" kind="definition" title="Big-O (asymptotic upper bound)">
$f \in O(g)$ if $\exists c > 0, n_0 \in \mathbb{N}$ such that $\forall n \ge n_0: f(n) \le c \cdot g(n)$.
</MathStatement>

Big-O bounds $f$ from above up to constant factors. For example, $f(n) = 3n + 5$ satisfies $f \in O(n)$ because $3n + 5 \le 4n$ for $n \ge 5$.

<MathStatement id="big-omega" kind="definition" title="Big-Omega (asymptotic lower bound)">
$f \in \Omega(g)$ if $\exists c > 0, n_0 \in \mathbb{N}$ such that $\forall n \ge n_0: f(n) \ge c \cdot g(n)$.
</MathStatement>

Big-Omega bounds $f$ from below. If $f(n) = n^2 - 100n$, then $f \in \Omega(n^2)$ because eventually $n^2 - 100n \ge 0.5 n^2$.

<MathStatement id="big-theta" kind="definition" title="Big-Theta (asymptotic tight bound)">
$f \in \Theta(g)$ if $f \in O(g)$ and $f \in \Omega(g)$.
</MathStatement>

Big-Theta captures the exact growth rate. For any polynomial $f(n) = a_k n^k + a_{k-1} n^{k-1} + \cdots + a_0$ with $a_k > 0$, we have $f \in \Theta(n^k)$.

<MathStatement id="little-o" kind="definition" title="Little-o (strict asymptotic upper bound)">
$f \in o(g)$ if $\forall c > 0, \exists n_0$ such that $\forall n \ge n_0: f(n) \le c \cdot g(n)$.
</MathStatement>

Little-o is strict: $f$ grows strictly slower than $g$. For any $k < m$, we have $n^k \in o(n^m)$. Unlike Big-O, $f \in o(f)$ is false.

<MathStatement id="little-omega" kind="definition" title="Little-omega (strict asymptotic lower bound)">
$f \in \omega(g)$ if $\forall c > 0, \exists n_0$ such that $\forall n \ge n_0: f(n) \ge c \cdot g(n)$.
</MathStatement>

Note the notational convention: $O(g)$ is a set of functions, so $f \in O(g)$ is precise. The common abuse $f = O(g)$ obscures transitivity and set membership, but appears widely in the literature.

## Limit Characterization

When the limit of the ratio exists, it directly determines the asymptotic class.

<MathStatement id="limit-char" kind="theorem" title="Limit characterization of Landau symbols">
Let $L = \lim_{n\to\infty} \frac{f(n)}{g(n)}$ exist (possibly infinite). Then:
- $0 < L < \infty \iff f \in \Theta(g)$,
- $L = 0 \iff f \in o(g)$,
- $L = \infty \iff f \in \omega(g)$.
</MathStatement>

**Proof.** If $0 < L < \infty$, then for sufficiently large $n$, $L/2 \le f(n)/g(n) \le 3L/2$, giving both $f \in O(g)$ and $f \in \Omega(g)$. If $L = 0$, for any $c > 0$, eventually $f(n)/g(n) < c$, so $f \in o(g)$. If $L = \infty$, for any $c > 0$, eventually $f(n)/g(n) > c$, so $f \in \omega(g)$. The converses follow from the definitions. □

When the limit does not exist, the $\limsup$ and $\liminf$ characterize the class:

- $f \in O(g) \iff \limsup_{n\to\infty} f(n)/g(n) < \infty$,
- $f \in \Omega(g) \iff \liminf_{n\to\infty} f(n)/g(n) > 0$,
- $f \in \Theta(g) \iff 0 < \liminf f/g \le \limsup f/g < \infty$.

**Example.** $f(n) = n(2 + \sin n)$ oscillates between $n$ and $3n$. $\liminf f/n = 1$, $\limsup f/n = 3$, both finite and nonzero, so $f \in \Theta(n)$.

## Properties of Asymptotic Classes

The Landau classes satisfy algebraic closure properties:

- **Additivity:** If $f_1 \in O(g_1)$ and $f_2 \in O(g_2)$, then $f_1 + f_2 \in O(g_1 + g_2) \subseteq O(\max(g_1, g_2))$.
- **Multiplication:** If $f_1 \in O(g_1)$ and $f_2 \in O(g_2)$, then $f_1 \cdot f_2 \in O(g_1 \cdot g_2)$.
- **Transitivity:** If $f \in O(g)$ and $g \in O(h)$, then $f \in O(h)$.
- **Reflexivity:** $f \in O(f)$ for any $f$.

These properties hold for $\Omega$ and $\Theta$ by symmetry. For $o$ and $\omega$, transitivity holds but reflexivity fails.

## The Asymptotic Hierarchy

The following functions form a strict hierarchy under little-o:

$$
1 \prec \log\log n \prec \log n \prec (\log n)^k \prec n^\epsilon \prec n \prec n\log n \prec n^{1+\epsilon} \prec n^2 \prec n^k \prec c^n \prec n! \prec n^n
$$

for any $k > 1$, $0 < \epsilon < 1$, and $c > 1$.

**Proof of the polynomial-logarithm gap.** For any $\epsilon > 0$ and integer $k$, $\log^k n \in o(n^\epsilon)$. Let $m = \log n$, so $n = e^m$. Then $\log^k n = m^k$ and $n^\epsilon = e^{\epsilon m}$. The ratio $m^k / e^{\epsilon m} \to 0$ as $m \to \infty$ because exponentials dominate any polynomial. □

**Proof of the exponential-polynomial gap.** For any $c > 1$ and integer $k$, $n^k \in o(c^n)$. The ratio $n^k / c^n \to 0$ because exponential growth outpaces any fixed-degree polynomial. □

## Common Modes of Analysis

Asymptotic notation applies in several analysis contexts:

- **Worst-case:** Maximum cost over all inputs of size $n$.
- **Average-case:** Expected cost over a specified input distribution.
- **Amortized:** Bound on the total cost of a sequence, divided by the number of operations.
- **Expected:** Cost bound that holds in expectation for randomized algorithms.

Each mode uses the same Landau notation but quantifies over different domains. A claim like "searching is $O(\log n)$" is meaningless without specifying which mode.

## Exercises

1. **Limit calculations.** Determine the asymptotic class ($O$, $\Omega$, $\Theta$, $o$, $\omega$) of each pair:
   - $f(n) = n^2 + 3n$, $g(n) = n^2$
   - $f(n) = \sqrt{n}$, $g(n) = n^{2/3}$
   - $f(n) = n \log n$, $g(n) = n \sqrt{n}$
   - $f(n) = 2^n$, $g(n) = 3^n$

2. **Non-limit.** Construct $f, g$ such that $f \in \Theta(g)$ but $\lim_{n\to\infty} f(n)/g(n)$ does not exist.

3. **Properties.** Prove or disprove: if $f \in O(g)$ and $g \in O(h)$, then $f + g \in O(h)$.

4. **Hierarchy placement.** Place each function in the hierarchy: $n/\log n$, $\log(n!)$, $n^{\log n}$, $2^{2^n}$.

<summary>Solutions</summary>

1. (a) $\Theta(n^2)$. (b) $o(\sqrt{n})$? $\sqrt{n} / n^{2/3} = n^{-1/6} \to 0$, so $f \in o(g)$. (c) $n\log n / (n\sqrt{n}) = \log n / \sqrt{n} \to 0$, so $f \in o(g)$. (d) $2^n / 3^n = (2/3)^n \to 0$, so $f \in o(g)$.

2. $f(n) = n$, $g(n) = n(2 + \sin n)$. Ratio oscillates between $1/3$ and $1$, never converging. Both $f \in O(g)$ and $g \in O(f)$ hold.

3. True. $f \in O(h)$ by transitivity. $g \in O(h)$ by assumption. So $f+g \in O(h+h) = O(h)$.

4. $n/\log n$ is between $n$ and $n\log n$. $\log(n!) \sim n\log n$, same as $n\log n$. $n^{\log n}$ is super-polynomial but sub-exponential. $2^{2^n}$ dominates everything.
