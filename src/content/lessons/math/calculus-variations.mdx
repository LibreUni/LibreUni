# Calculus of Variations

In standard calculus, we seek a *number* $x$ that minimizes or maximizes a function $f(x)$. In the Calculus of Variations, we seek a *function* $y(x)$ that minimizes or maximizes a **functional** $J[y]$ — which is a mapping from a vector space of functions to the real numbers.

---

## Functionals and the Euler-Lagrange Equation

Let $y(x)$ be a continuously differentiable function on $[x_1, x_2]$ with fixed boundary conditions $y(x_1) = y_1$ and $y(x_2) = y_2$.

<MathStatement id="functional-def" kind="definition" title="Functional" number="1">

A **functional** $J[y]$ is defined by an integral of the form:

$$J[y] = \int_{x_1}^{x_2} L(x, y(x), y'(x)) \, dx$$

where the integrand $L(x, y, y')$ is a smooth function called the **Lagrangian**.

</MathStatement>

We want to find the function $y(x)$ that makes $J[y]$ stationary. We do this by perturbing $y(x)$ by a small test function $\eta(x)$ satisfying $\eta(x_1) = \eta(x_2) = 0$.

<MathStatement id="euler-lagrange" kind="theorem" title="The Euler-Lagrange Equation" number="1" proofId="euler-lagrange-proof">

If $y(x)$ is a continuously differentiable function that extremizes the functional $J[y]$ subject to fixed boundary conditions, then $y(x)$ must satisfy the **Euler-Lagrange equation**:

$$\frac{\partial L}{\partial y} - \frac{d}{dx} \left( \frac{\partial L}{\partial y'} \right) = 0.$$

</MathStatement>

<MathProof id="euler-lagrange-proof" for="euler-lagrange">

Let $y(x)$ be the extremizer. We define a family of perturbed functions $Y(x) = y(x) + \epsilon \eta(x)$, where $\epsilon \in \mathbb{R}$ is a small parameter, and $\eta(x)$ is a smooth function with $\eta(x_1) = \eta(x_2) = 0$.
The functional evaluated along this family is a function of $\epsilon$:

$$\Phi(\epsilon) = J[y + \epsilon \eta] = \int_{x_1}^{x_2} L(x, y + \epsilon \eta, y' + \epsilon \eta') \, dx.$$

Since $y(x)$ is an extremizer, $\Phi(\epsilon)$ must have a local extremum at $\epsilon = 0$. Thus, $\Phi'(0) = 0$.
Differentiating under the integral sign:

$$\Phi'(\epsilon) = \int_{x_1}^{x_2} \left( \frac{\partial L}{\partial y} \frac{\partial Y}{\partial \epsilon} + \frac{\partial L}{\partial y'} \frac{\partial Y'}{\partial \epsilon} \right) \, dx.$$

Since $\frac{\partial Y}{\partial \epsilon} = \eta(x)$ and $\frac{\partial Y'}{\partial \epsilon} = \eta'(x)$:

$$\Phi'(\epsilon) = \int_{x_1}^{x_2} \left( \frac{\partial L}{\partial y} \eta(x) + \frac{\partial L}{\partial y'} \eta'(x) \right) \, dx.$$

Setting $\epsilon = 0$ yields:

$$0 = \int_{x_1}^{x_2} \frac{\partial L}{\partial y} \eta(x) \, dx + \int_{x_1}^{x_2} \frac{\partial L}{\partial y'} \eta'(x) \, dx.$$

Apply integration by parts to the second term:

$$\int_{x_1}^{x_2} \frac{\partial L}{\partial y'} \eta'(x) \, dx = \left[ \frac{\partial L}{\partial y'} \eta(x) \right]_{x_1}^{x_2} - \int_{x_1}^{x_2} \frac{d}{dx}\left(\frac{\partial L}{\partial y'}\right) \eta(x) \, dx.$$

Since $\eta(x_1) = \eta(x_2) = 0$, the boundary term vanishes. Substituting this back into the equation:

$$0 = \int_{x_1}^{x_2} \left( \frac{\partial L}{\partial y} - \frac{d}{dx}\left(\frac{\partial L}{\partial y'}\right) \right) \eta(x) \, dx.$$

Since this integral must equal 0 for *every* smooth test function $\eta(x)$, by the **Fundamental Lemma of the Calculus of Variations**, the integrand itself must be zero everywhere:

$$\frac{\partial L}{\partial y} - \frac{d}{dx}\left(\frac{\partial L}{\partial y'}\right) = 0.$$

This completes the proof.

</MathProof>

## Famous Classical Problems

### 1. Geodesics (Shortest Path in $\mathbb{R}^2$)

The length of a curve $y(x)$ from $x_1$ to $x_2$ is:

$$J[y] = \int_{x_1}^{x_2} \sqrt{1 + (y')^2} \, dx.$$

Here $L(x, y, y') = \sqrt{1 + (y')^2}$. Since $\frac{\partial L}{\partial y} = 0$, the Euler-Lagrange equation becomes:

$$\frac{d}{dx} \left( \frac{y'}{\sqrt{1 + (y')^2}} \right) = 0 \implies \frac{y'}{\sqrt{1 + (y')^2}} = C \implies y' = m \text{ (constant)}.$$

Thus, $y(x) = mx + b$, proving the shortest path between two points in Euclidean space is a straight line.

### 2. The Brachistochrone

The shape of a wire connecting $A$ and $B$ that minimizes travel time for a bead sliding under gravity is a **cycloid**. The functional is derived using conservation of energy $v = \sqrt{2gy}$:

$$T[y] = \int_{x_1}^{x_2} \frac{ds}{v} = \int_{x_1}^{x_2} \frac{\sqrt{1 + (y')^2}}{\sqrt{2gy}} \, dx.$$

## Hamilton's Principle of Least Action

In classical mechanics, the path a physical system takes is the one that minimizes the **Action** $S$:

$$S = \int_{t_1}^{t_2} (T - V) \, dt$$

where $T$ is the kinetic energy and $V$ is the potential energy. The Lagrangian is $L = T - V$. The Euler-Lagrange equations yield **Newton's Second Law**: $\mathbf{F} = m\mathbf{a}$.

Let us write a Python script using gradient-based optimization to solve for the shortest path (geodesic) between $(0, 0)$ and $(1, 2)$ in the plane by discretizing a path and minimizing its length numerically.

<CodeRunner client:load
  code={`import numpy as np
from scipy.optimize import minimize

# Shortest path between (0,0) and (1,2)
# Discretize the path into N segments
N = 10
x = np.linspace(0.0, 1.0, N)
# Initial guess: a curved path
y_guess = 2.0 * x**2

def path_length(y_internal):
    # Reconstruct full y including boundary points y(0)=0 and y(1)=2
    y_full = np.concatenate(([0.0], y_internal, [2.0]))
    dy = np.diff(y_full)
    dx = x[1] - x[0]
    lengths = np.sqrt(dx**2 + dy**2)
    return np.sum(lengths)

# Optimize internal y values (8 points)
res = minimize(path_length, y_guess[1:-1])

y_opt = np.concatenate(([0.0], res.x, [2.0]))
print("Optimized Y values:", np.round(y_opt, 4))
print("Linear interpolation target:", np.round(2.0 * x, 4))
print(f"Path length: {res.fun:.6f} | Straight line length: {np.sqrt(1**2 + 2**2):.6f}")
`}
