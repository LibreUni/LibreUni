# Complex Analysis

Complex analysis is the study of functions of a complex variable $z = x + iy$. The transition from real analysis to complex analysis reveals a profound rigidity: while a real function can be differentiable once without having any higher derivatives, a complex function that is differentiable once in a neighborhood is automatically infinitely differentiable and analytic.

---

## Complex Differentiability and the Cauchy-Riemann Equations

Let $f: U \subseteq \mathbb{C} \to \mathbb{C}$ be a function defined on an open set $U$.

<MathStatement id="complex-differentiability" kind="definition" title="Complex Differentiability and Holomorphy" number="1">

A function $f$ is **complex-differentiable** at $z_0 \in U$ if the limit:

$$f'(z_0) = \lim_{z \to z_0} \frac{f(z) - f(z_0)}{z - z_0}$$

exists. If $f$ is complex-differentiable at every point in a neighborhood of $z_0$, we say $f$ is **holomorphic** (or analytic) at $z_0$.

</MathStatement>

Letting $f(z) = u(x, y) + i v(x, y)$ and $z = x + iy$, the existence of this limit along different directions forces a strict relation between the partial derivatives of the real and imaginary parts.

Complex differentiability requires the same difference quotient limit from every direction in the complex plane:

<MathStatement id="cauchy-riemann" kind="theorem" title="Cauchy-Riemann Equations" number="1" proofId="cauchy-riemann-proof">

If $f(z) = u(x, y) + iv(x, y)$ is complex-differentiable at $z_0 = x_0 + i y_0$, then the first-order partial derivatives of $u$ and $v$ exist at $(x_0, y_0)$ and satisfy:

$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y} \quad \text{and} \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}.$$

</MathStatement>

<MathProof id="cauchy-riemann-proof" for="cauchy-riemann">

We compute the limit of the difference quotient in two different directions.
Let $h \in \mathbb{R}$ be real.
1. **Horizontal Approach**: Let $z \to z_0$ along the horizontal line $z = z_0 + h$:
$$f'(z_0) = \lim_{h \to 0} \frac{u(x_0+h, y_0) + i v(x_0+h, y_0) - (u(x_0, y_0) + iv(x_0, y_0))}{h}$$
$$= \lim_{h \to 0} \frac{u(x_0+h, y_0) - u(x_0, y_0)}{h} + i \lim_{h \to 0} \frac{v(x_0+h, y_0) - v(x_0, y_0)}{h}$$
$$= \frac{\partial u}{\partial x}(x_0, y_0) + i \frac{\partial v}{\partial x}(x_0, y_0).$$

2. **Vertical Approach**: Let $z \to z_0$ along the vertical line $z = z_0 + ik$:
$$f'(z_0) = \lim_{k \to 0} \frac{u(x_0, y_0+k) + i v(x_0, y_0+k) - (u(x_0, y_0) + iv(x_0, y_0))}{ik}$$
$$= \lim_{k \to 0} \frac{1}{i} \frac{u(x_0, y_0+k) - u(x_0, y_0)}{k} + \lim_{k \to 0} \frac{v(x_0, y_0+k) - v(x_0, y_0)}{k}.$$

Since $\frac{1}{i} = -i$, this simplifies to:

$$f'(z_0) = \frac{\partial v}{\partial y}(x_0, y_0) - i \frac{\partial u}{\partial y}(x_0, y_0).$$

Since the limit must be unique, we equate the real and imaginary parts of the horizontal and vertical derivations:

$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y} \quad \text{and} \quad \frac{\partial v}{\partial x} = -\frac{\partial u}{\partial y} \iff \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}.$$

This completes the proof.

</MathProof>

## Cauchy's Integral Theorem and Formula

Holomorphic functions have exceptional boundary integration properties.

<MathStatement id="cauchys-integral-theorem" kind="theorem" title="Cauchy's Integral Theorem" number="2">

If $f$ is holomorphic in a simply connected open set $U$, and $\gamma$ is a closed rectifiable curve in $U$, then:

$$\oint_\gamma f(z) \, dz = 0.$$

</MathStatement>

This theorem implies that the value of a holomorphic function inside a region is completely determined by its values on the boundary curve.

<MathStatement id="cauchys-integral-formula" kind="theorem" title="Cauchy's Integral Formula" number="3">

Let $f$ be holomorphic in an open set containing the closed disk $D = \{z \mid |z - z_0| \le R\}$. Let $\gamma$ be the boundary circle of $D$ oriented positively. Then for any $a$ in the interior of $D$:

$$f(a) = \frac{1}{2\pi i} \oint_\gamma \frac{f(z)}{z - a} \, dz.$$

More generally, the $n$-th derivative is:

$$f^{(n)}(a) = \frac{n!}{2\pi i} \oint_\gamma \frac{f(z)}{(z - a)^{n+1}} \, dz.$$

</MathStatement>

## Laurent Series and the Residue Theorem

Near a singularity (a point where a function is not holomorphic), we can expand a function in both positive and negative powers.

<MathStatement id="laurent-series" kind="definition" title="Laurent Series and Singularities" number="2">

A **Laurent series** for a function $f(z)$ about $z_0$ is:

$$f(z) = \sum_{n=-\infty}^\infty c_n (z - z_0)^n.$$

The coefficient $c_{-1}$ is the **residue** of $f$ at $z_0$, denoted $\operatorname{Res}(f, z_0)$.
The nature of the singularity at $z_0$ is classified by the negative terms:
1. **Removable singularity**: No negative power terms.
2. **Pole of order $m$**: Finitely many negative terms up to $n = -m$.
3. **Essential singularity**: Infinitely many negative power terms.

</MathStatement>

If we integrate a function around a closed path containing singularities, the integral is determined entirely by the residues.

<MathStatement id="residue-theorem" kind="theorem" title="Cauchy's Residue Theorem" number="4">

Let $U$ be a simply connected open set, and let $\gamma$ be a positively oriented simple closed path in $U$. If $f$ is holomorphic on $U$ except at a finite number of points $z_1, \dots, z_k$ inside $\gamma$, then:

$$\oint_\gamma f(z) \, dz = 2\pi i \sum_{j=1}^k \operatorname{Res}(f, z_j).$$

</MathStatement>

Let us write a Python script using complex arithmetic to verify that the derivative of $f(z) = z^2$ is independent of the limit direction (horizontal vs. vertical), proving it is holomorphic, whereas $f(z) = \bar{z}$ is not.

<CodeRunner client:load
  code={`def complex_diff(f, z, h=1e-7):
    # Limit from horizontal direction
    df_horiz = (f(z + h) - f(z)) / h
    # Limit from vertical direction
    df_vert = (f(z + h*1j) - f(z)) / (h*1j)
    return df_horiz, df_vert

# Test f(z) = z^2 (holomorphic)
f_holo = lambda z: z**2
z_pt = 1.0 + 1.0j
horiz, vert = complex_diff(f_holo, z_pt)
print(f"z^2 Horiz derivative: {horiz}")
print(f"z^2 Vert derivative:  {vert}")
print(f"Holomorphic? {abs(horiz - vert) < 1e-5}\\n")

# Test f(z) = conj(z) (not holomorphic)
f_nonholo = lambda z: z.conjugate()
horiz, vert = complex_diff(f_nonholo, z_pt)
print(f"conj(z) Horiz derivative: {horiz}")
print(f"conj(z) Vert derivative:  {vert}")
print(f"Holomorphic? {abs(horiz - vert) < 1e-5}")
`}
