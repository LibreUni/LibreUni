# Ordinary Differential Equations

Ordinary Differential Equations (ODEs) relate an unknown function of a single variable to its derivatives. They form the foundational mathematical language of physics, engineering, and chemistry, describing how physical states evolve over time.

---

## Classification of Ordinary Differential Equations

Let $y$ be a function of $x$.

<MathStatement id="ode-classification" kind="definition" title="Order and Linearity" number="1">

An **ordinary differential equation (ODE)** of order $n$ is an equation of the form:

$$F(x, y, y', y'', \dots, y^{(n)}) = 0.$$

- The **order** of the ODE is the order of the highest derivative present in the equation.
- The ODE is **linear** if it can be written in the form:
$$a_n(x) y^{(n)} + a_{n-1}(x) y^{(n-1)} + \dots + a_1(x) y' + a_0(x) y = g(x).$$
If $g(x) = 0$, the linear equation is **homogeneous**; otherwise, it is **non-homogeneous**.

</MathStatement>

---

## First-Order Analytical Solution Methods

We review three key classes of first-order ODEs that can be solved analytically.

### 1. Separable Equations

If an equation can be written as $y' = g(x)h(y)$, we separate variables and integrate:

$$\frac{1}{h(y)} \, dy = g(x) \, dx \implies \int \frac{1}{h(y)} \, dy = \int g(x) \, dx.$$

### 2. First-Order Linear Equations (Integrating Factors)

An equation of the form $y' + P(x)y = Q(x)$ is solved by multiplying both sides by the **integrating factor** $I(x) = e^{\int P(x) \, dx}$:

$$\frac{d}{dx}[I(x)y] = I(x)Q(x) \implies y(x) = \frac{1}{I(x)} \left( \int I(x)Q(x) \, dx + C \right).$$

### 3. Exact Equations

An equation $M(x, y) \, dx + N(x, y) \, dy = 0$ is **exact** if there exists a potential function $\Psi(x, y)$ such that $\frac{\partial \Psi}{\partial x} = M$ and $\frac{\partial \Psi}{\partial y} = N$. By Clairaut's Theorem, this is equivalent to:

$$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}.$$

---

## Existence and Uniqueness

Before solving an ODE numerically, we must verify that a unique solution exists in the neighborhood of the initial value.

<MathStatement id="picard-theorem" kind="theorem" title="Picard-Lindelöf Existence and Uniqueness Theorem" number="1">

Let $f(x, y)$ be continuous on a closed rectangle $R = [x_0 - a, x_0 + a] \times [y_0 - b, y_0 + b]$ and satisfy a **Lipschitz condition** with respect to $y$:

$$|f(x, y_1) - f(x, y_2)| \le L |y_1 - y_2| \quad \text{for all } (x, y_1), (x, y_2) \in R.$$

Then the initial value problem:

$$y' = f(x, y), \quad y(x_0) = y_0$$

has a unique solution $y(x)$ in some interval $[x_0 - h, x_0 + h]$ where $h > 0$.

</MathStatement>

---

## Numerical Integration Methods

When analytical solutions are unavailable, we discretize the domain $x$ with step size $h$ and compute numerical approximations.

- **Euler's Method (First-Order)**:
$$y_{n+1} = y_n + h f(x_n, y_n).$$
- **Runge-Kutta 4th Order (RK4 - Fourth-Order)**:
$$k_1 = f(x_n, y_n), \quad k_2 = f\left(x_n + \frac{h}{2}, y_n + h\frac{k_1}{2}\right)$$
$$k_3 = f\left(x_n + \frac{h}{2}, y_n + h\frac{k_2}{2}\right), \quad k_4 = f(x_n + h, y_n + h k_3)$$
$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4).$$

Let us write a Python script to compare the accuracy of Euler's method and the RK4 method in solving the IVP $y' = -y + x$ with $y(0) = 1.0$ up to $x = 2.0$. The exact analytical solution is $y(x) = 2e^{-x} + x - 1$.

<CodeRunner client:load
  code={`import numpy as np

def f(x, y):
    return -y + x

def exact_sol(x):
    return 2.0 * np.exp(-x) + x - 1.0

x0, y0 = 0.0, 1.0
x_target = 2.0
h = 0.5 # Large step size to emphasize error difference
steps = int((x_target - x0) / h)

# Euler Solver
x_euler, y_euler = x0, y0
for _ in range(steps):
    y_euler = y_euler + h * f(x_euler, y_euler)
    x_euler += h

# RK4 Solver
x_rk, y_rk = x0, y0
for _ in range(steps):
    k1 = f(x_rk, y_rk)
    k2 = f(x_rk + h/2, y_rk + h*k1/2)
    k3 = f(x_rk + h/2, y_rk + h*k2/2)
    k4 = f(x_rk + h, y_rk + h*k3)
    y_rk = y_rk + (h/6) * (k1 + 2*k2 + 2*k3 + k4)
    x_rk += h

exact = exact_sol(x_target)
print(f"Exact Solution at x=2.0: {exact:.10f}")
print(f"Euler Solution:          {y_euler:.10f} | Error: {abs(y_euler - exact):.10f}")
print(f"RK4 Solution:            {y_rk:.10f} | Error: {abs(y_rk - exact):.10f}")
`}
/>
