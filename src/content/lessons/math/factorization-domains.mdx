# Factorization in Integral Domains

In the integers $\mathbb{Z}$, every number can be uniquely factored into a product of prime numbers (e.g., $12 = 2^2 \cdot 3$). This is the Fundamental Theorem of Arithmetic. But does this unique factorization hold in all rings where we can add and multiply?

The answer is **no**. In many rings, elements can be factored into "indivisible" pieces in completely different ways, and the definition of a "prime" becomes more subtle. To study where factorization works, we classify domains into a hierarchy: **ED $\implies$ PID $\implies$ UFD**.

## The Motivating Problem: Factorization Failure in $\mathbb{Z}[\sqrt{-5}]$

Consider the ring:
$$\mathbb{Z}[\sqrt{-5}] = \{ a + b\sqrt{-5} \mid a, b \in \mathbb{Z} \}$$

Let's look at the number $6$ in this ring. We can write $6$ in two different ways:
- **First factorization:** $6 = 2 \cdot 3$
- **Second factorization:** $6 = (1 + \sqrt{-5})(1 - \sqrt{-5})$

Are these factorizations actually different? To check this, we must verify if the elements $2, 3, 1+\sqrt{-5},$ and $1-\sqrt{-5}$ can be factored any further in $\mathbb{Z}[\sqrt{-5}]$, or if they are "indivisible" (irreducible). 

Using the norm function $N(a + b\sqrt{-5}) = a^2 + 5b^2$, we can prove that all four of these elements are indeed irreducible, and they are not simply multiples of each other by units (which are only $1$ and $-1$). Thus, the number 6 has two completely distinct factorizations into irreducible elements!

### Try It Yourself: Norm Calculations

The norm function is multiplicative: $N(\alpha \beta) = N(\alpha)N(\beta)$.
Let's compute the norm of $1 + \sqrt{-5}$:
$$N(1 + \sqrt{-5}) = 1^2 + 5(1)^2 = 6$$

If $1 + \sqrt{-5} = \alpha \beta$, then $N(\alpha)N(\beta) = 6$. 
Since the norm of any element $a + b\sqrt{-5}$ is $a^2 + 5b^2$, what are the possible integer values that the norm of an element in $\mathbb{Z}[\sqrt{-5}]$ can take?

## Formalization: Irreducibles vs Primes

Let $R$ be an integral domain, and let $p \in R$ be a non-zero element that is not a unit.
- $p$ is **irreducible** if whenever $p = ab$ for some $a, b \in R$, then either $a$ or $b$ is a unit (meaning $p$ cannot be factored into two non-trivial pieces).
- $p$ is a **prime** if whenever $p \mid ab$ for some $a, b \in R$, then either $p \mid a$ or $p \mid b$.

> **Theorem:** In any integral domain, every prime element is irreducible. However, irreducible elements are not necessarily prime. (In $\mathbb{Z}[\sqrt{-5}]$, the element $2$ is irreducible but not prime, since $2 \mid (1+\sqrt{-5})(1-\sqrt{-5}) = 6$ but $2$ does not divide either factor).

## The Factorization Hierarchy

To ensure unique factorization, we define three classes of domains:

1. **Unique Factorization Domain (UFD):** An integral domain in which every non-zero non-unit element can be factored into a product of irreducibles, and this factorization is unique up to order and multiplication by units.
   *Theorem:* In a UFD, an element is prime if and only if it is irreducible.
2. **Principal Ideal Domain (PID):** An integral domain in which every ideal is principal (generated by a single element).
   *Theorem:* Every PID is a UFD.
3. **Euclidean Domain (ED):** An integral domain equipped with a Euclidean function $N: R \setminus \{0\} \to \mathbb{Z}_{\ge 0}$ that allows for division with remainder: for all $a, b \in R$ with $b \neq 0$, there exist $q, r \in R$ such that:
   $$a = qb + r \quad \text{with} \quad r = 0 \text{ or } N(r) < N(b)$$
   *Theorem:* Every ED is a PID.

The hierarchy is: **ED $\implies$ PID $\implies$ UFD $\implies$ Integral Domain**.

## Worked Example: Division in the Gaussian Integers $\mathbb{Z}[i]$

The ring of Gaussian Integers $\mathbb{Z}[i] = \{ a + bi \mid a, b \in \mathbb{Z} \}$ is a Euclidean Domain with the norm function $N(a + bi) = a^2 + b^2$. 
Let's divide $\alpha = 5 + 3i$ by $\beta = 2 - i$.
1. Compute the fraction in $\mathbb{C}$:
   $$\frac{5 + 3i}{2 - i} = \frac{(5 + 3i)(2 + i)}{(2 - i)(2 + i)} = \frac{10 + 5i + 6i - 3}{5} = \frac{7 + 11i}{5} = 1.4 + 2.2i$$
2. Round the coefficients to the nearest integers to find the quotient $q$:
   $$q = 1 + 2i$$
3. Compute the remainder $r$:
   $$r = \alpha - q\beta = (5 + 3i) - (1 + 2i)(2 - i) = (5 + 3i) - (4 + 3i) = 1$$
4. Check the norm of $r$:
   $$N(r) = N(1) = 1 < N(\beta) = N(2-i) = 5$$
So $5 + 3i = (1 + 2i)(2 - i) + 1$, and $N(1) < N(2-i)$. The division works!

Let's write a Python script to execute the Euclidean Algorithm on Gaussian integers to find their Greatest Common Divisor (GCD).

## Contrast: PID vs UFD

- **$\mathbb{Z}[x]$ (Polynomials with integer coefficients):** This ring is a UFD (by Gauss's Lemma), meaning every polynomial factors uniquely into irreducibles. However, it is **NOT** a PID. 
  For example, the ideal $I = \langle 2, x \rangle$ (all polynomials with even constant terms) cannot be generated by a single polynomial.
- **$\mathbb{Z}$ (Integers):** Is both a PID and a UFD (and also an ED).
