# Fourier Analysis

Fourier Analysis is the mathematical framework for representing complex, periodic signals as linear combinations of simple sinusoidal waves (Fourier series), and generalizing this to non-periodic signals using integral transformations (Fourier transform).

---

## Trigonometric Fourier Series

Let $f(x)$ be a periodic function of period $2L$.

<MathStatement id="fourier-series" kind="definition" title="Fourier Series and Coefficients" number="1">

The **Fourier series** of a periodic function $f(x)$ with period $2L$ on the interval $[-L, L]$ is defined as:

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty \left( a_n \cos\left(\frac{n\pi x}{L}\right) + b_n \sin\left(\frac{n\pi x}{L}\right) \right)$$

where the **Fourier coefficients** $a_n$ and $b_n$ are given by the Euler-Fourier formulas:

$$a_n = \frac{1}{L} \int_{-L}^L f(x) \cos\left(\frac{n\pi x}{L}\right) \, dx \quad \text{for } n \ge 0,$$
$$b_n = \frac{1}{L} \int_{-L}^L f(x) \sin\left(\frac{n\pi x}{L}\right) \, dx \quad \text{for } n \ge 1.$$

</MathStatement>

Fourier analysis separates a complicated periodic signal into simpler frequencies:

For the series to converge exactly to the function, the function must satisfy regularity conditions.

<MathStatement id="dirichlet-conditions" kind="theorem" title="Dirichlet's Convergence Theorem" number="1">

If $f(x)$ is a periodic function of period $2L$ that is piecewise continuous and piecewise smooth on $[-L, L]$, then the Fourier series converges to:
1. $f(x)$ at any point $x$ where $f$ is continuous.
2. $\frac{f(x^+) + f(x^-)}{2}$ at any point $x$ where $f$ has a jump discontinuity.

</MathStatement>

## Complex Fourier Series

Using Euler's formula $e^{i\theta} = \cos\theta + i\sin\theta$, we can represent the Fourier series more compactly using complex exponentials.

<MathStatement id="complex-fourier" kind="definition" title="Complex Fourier Series" number="2">

The **complex Fourier series** of $f(x)$ with period $2L$ is:

$$f(x) \sim \sum_{n=-\infty}^\infty c_n e^{i \frac{n\pi x}{L}}$$

where the complex coefficients $c_n$ are defined as:

$$c_n = \frac{1}{2L} \int_{-L}^L f(x) e^{-i \frac{n\pi x}{L}} \, dx.$$

The real and complex coefficients are related by: $c_n = \frac{a_n - i b_n}{2}$ and $c_{-n} = \frac{a_n + i b_n}{2}$ for $n \ge 1$.

</MathStatement>

## The Fourier Transform

For non-periodic functions, we take the limit of the period $L \to \infty$. This transitions the discrete sum over frequencies into a continuous integral.

<MathStatement id="fourier-transform" kind="definition" title="Fourier Transform" number="3">

Let $f: \mathbb{R} \to \mathbb{C}$ be an integrable function ($f \in L^1(\mathbb{R})$). The **Fourier transform** of $f$, denoted $\hat{f}(\xi)$ or $\mathcal{F}\{f\}$, is defined as:

$$\hat{f}(\xi) = \int_{-\infty}^\infty f(x) e^{-2\pi i x \xi} \, dx \quad \text{for all } \xi \in \mathbb{R}.$$

The **Inverse Fourier transform** reconstructs the original function from its frequency spectrum:

$$f(x) = \int_{-\infty}^\infty \hat{f}(\xi) e^{2\pi i x \xi} \, d\xi.$$

</MathStatement>

## Parseval's and Plancherel's Theorems

Fourier transformations preserve the $L^2$-norm (total energy) of a signal.

<MathStatement id="parsevals-theorem" kind="theorem" title="Parseval's and Plancherel's Theorems" number="2">

1. **Parseval's Identity (Fourier Series)**: If $f$ is square-integrable on $[-L, L]$:
$$\frac{1}{2L} \int_{-L}^L |f(x)|^2 \, dx = \sum_{n=-\infty}^\infty |c_n|^2.$$
2. **Plancherel's Theorem (Fourier Transform)**: If $f \in L^2(\mathbb{R})$:
$$\int_{-\infty}^\infty |f(x)|^2 \, dx = \int_{-\infty}^\infty |\hat{f}(\xi)|^2 \, d\xi.$$

</MathStatement>

Let us write a Python script using the Fast Fourier Transform (FFT) to analyze the frequency components of a signal composed of two sine waves at $5\text{ Hz}$ and $12\text{ Hz}$ mixed with noise.

<CodeRunner client:load
  code={`import numpy as np

# Sampling parameters
fs = 100.0          # Sampling rate (Hz)
t = np.arange(0, 1.0, 1.0/fs) # 1 second of data
signal = np.sin(2 * np.pi * 5.0 * t) + 0.5 * np.sin(2 * np.pi * 12.0 * t)

# Compute FFT
fft_vals = np.fft.fft(signal)
fft_freqs = np.fft.fftfreq(len(signal), 1.0/fs)

# Get positive frequency indices
pos_idx = np.where(fft_freqs >= 0)
freqs = fft_freqs[pos_idx]
magnitudes = np.abs(fft_vals)[pos_idx] / len(signal) # Normalize

# Print top frequencies with largest magnitudes
top_indices = np.argsort(magnitudes)[::-1][:3]
for idx in top_indices:
    if magnitudes[idx] > 0.05:
        print(f"Detected frequency: {freqs[idx]:.1f} Hz | Magnitude: {magnitudes[idx]:.4f}")
`}
