# The Fundamental Theorem of Calculus

The Fundamental Theorem of Calculus (FTC) is the central theorem of calculus, establishing the inverse relationship between differentiation (slopes) and integration (accumulation).

---

## FTC Part 1: Derivative of the Accumulation Function

This part of the theorem states that the rate of change of the area under a curve is equal to the height of the curve.

<MathStatement id="ftc-part-1" kind="theorem" title="Fundamental Theorem of Calculus, Part 1" number="1" proofId="ftc-part-1-proof">

Let $f: [a, b] \to \mathbb{R}$ be a Riemann integrable function, and let $f$ be continuous on $[a, b]$. Define the accumulation function $F: [a, b] \to \mathbb{R}$ by:

$$F(x) = \int_a^x f(t) \, dt.$$

Then $F$ is continuous on $[a, b]$, differentiable on $(a, b)$, and:

$$F'(x) = f(x) \quad \text{for all } x \in (a, b).$$

</MathStatement>

<MathProof id="ftc-part-1-proof" for="ftc-part-1">

We prove $F'(x) = f(x)$ at any $x \in (a, b)$. Using the limit definition of the derivative:

$$F'(x) = \lim_{h \to 0} \frac{F(x+h) - F(x)}{h}.$$

Using the additivity property of the Riemann integral, $F(x+h) - F(x) = \int_x^{x+h} f(t) \, dt$. Thus:

$$\frac{F(x+h) - F(x)}{h} = \frac{1}{h} \int_x^{x+h} f(t) \, dt.$$

Since $f$ is continuous on the interval $[x, x+h]$ (or $[x+h, x]$ if $h < 0$), by the Mean Value Theorem for Integrals, there exists some $c_h$ between $x$ and $x+h$ such that:

$$\int_x^{x+h} f(t) \, dt = f(c_h) \cdot h \implies \frac{1}{h} \int_x^{x+h} f(t) \, dt = f(c_h).$$

Now take the limit as $h \to 0$. Since $c_h$ is trapped between $x$ and $x+h$, by the Squeeze Theorem, $\lim_{h \to 0} c_h = x$.
Since $f$ is continuous, we have:

$$F'(x) = \lim_{h \to 0} f(c_h) = f\left(\lim_{h \to 0} c_h\right) = f(x).$$

This completes the proof.

</MathProof>

## FTC Part 2: Evaluation of Definite Integrals

This part provides the algebraic shortcut to evaluate definite integrals using antiderivatives.

<MathStatement id="ftc-part-2" kind="theorem" title="Fundamental Theorem of Calculus, Part 2" number="2" proofId="ftc-part-2-proof" dependsOn={[{ id: "ftc-part-1", label: "FTC Part 1" }]}>

Let $f: [a, b] \to \mathbb{R}$ be continuous on $[a, b]$. If $G$ is any antiderivative of $f$ on $[a, b]$ (that is, $G'(x) = f(x)$ for all $x \in [a, b]$), then:

$$\int_a^b f(x) \, dx = G(b) - G(a).$$

</MathStatement>

<MathProof id="ftc-part-2-proof" for="ftc-part-2">

Let $F(x) = \int_a^x f(t) \, dt$. By [FTC Part 1](#ftc-part-1), $F'(x) = f(x)$.
Since $G$ is also an antiderivative of $f$, we have $G'(x) = f(x) = F'(x)$ for all $x \in [a, b]$.
Therefore, $(F - G)'(x) = 0$ for all $x \in [a, b]$.
By the Mean Value Theorem Corollary, there exists a constant $C$ such that:

$$F(x) = G(x) + C \quad \text{for all } x \in [a, b].$$

To find $C$, we evaluate at $x = a$:

$$F(a) = G(a) + C \implies \int_a^a f(t) \, dt = G(a) + C \implies 0 = G(a) + C \implies C = -G(a).$$

Substituting $C$ back into the equation:

$$F(x) = G(x) - G(a).$$

Now, evaluating at $x = b$:

$$F(b) = G(b) - G(a) \implies \int_a^b f(t) \, dt = G(b) - G(a).$$

This completes the proof.

</MathProof>

## Integration Techniques from Derivative Rules

By reversing derivative rules, we obtain powerful integration tools.

### Substitution Rule (Reverse Chain Rule)

If $u = g(x)$ is differentiable and $f$ is continuous, then:

$$\int f(g(x))g'(x) \, dx = \int f(u) \, du.$$

### Integration by Parts (Reverse Product Rule)

If $u(x)$ and $v(x)$ are differentiable, then:

$$\int u \, dv = uv - \int v \, du.$$

Let us use SymPy to find the antiderivative of $x \ln(x)$ using integration by parts, and evaluate its definite integral from $1$ to $e$.

<CodeRunner client:load
  code={`import sympy as sp

x = sp.Symbol('x')
f = x * sp.log(x)

# Antiderivative (Indefinite integral)
F = sp.integrate(f, x)
print(f"Indefinite integral of x*ln(x): {F}")

# Definite integral from 1 to e
a, b = 1, sp.E
definite_integral = sp.integrate(f, (x, a, b))
print(f"Definite integral from 1 to e: {definite_integral}")
print(f"Numerical value: {float(definite_integral):.6f}")
`}
