# Galois Groups and the Fundamental Theorem

When we adjoin the roots of a polynomial to a field, the resulting extension has symmetries: automorphisms that permute the roots while fixing the base field. The group of these symmetries — the **Galois group** — reveals the internal structure of the extension.

## The Motivating Problem: Automorphisms of $\mathbb{Q}(\sqrt{2})$

Let $L = \mathbb{Q}(\sqrt{2})$. An **automorphism** of $L$ is a bijective ring homomorphism $\sigma: L \to L$. Since $\sigma$ fixes $\mathbb{Q}$ pointwise and must preserve arithmetic:
$$\sigma(\sqrt{2})^2 = \sigma(\sqrt{2} \cdot \sqrt{2}) = \sigma(2) = 2$$

So $\sigma(\sqrt{2})$ must be a number in $L$ whose square is $2$, giving two possibilities:
$$\sigma(\sqrt{2}) = \sqrt{2} \quad \text{or} \quad \sigma(\sqrt{2}) = -\sqrt{2}$$

These two automorphisms form a group of order 2 under composition — the **Galois group** $\text{Gal}(\mathbb{Q}(\sqrt{2})/\mathbb{Q})$.

### Automorphisms of $\mathbb{Q}(\sqrt[3]{2})$

Now consider $L = \mathbb{Q}(\sqrt[3]{2})$. The element $\sqrt[3]{2}$ is a root of $x^3 - 2$, whose other roots are $\sqrt[3]{2}\omega$ and $\sqrt[3]{2}\omega^2$ where $\omega = e^{2\pi i/3}$. But $L$ lies inside $\mathbb{R}$, so these complex roots are not in $L$.

## Formalization: Galois Groups

An extension $L/K$ is a **Galois extension** if it is both normal (a splitting field) and separable.

The **Galois group** $\text{Gal}(L/K)$ is the group of all field automorphisms of $L$ that fix $K$ pointwise:
$$\text{Gal}(L/K) = \{ \sigma: L \to L \mid \sigma \text{ automorphism}, \sigma(k) = k \text{ for all } k \in K \}$$

> **Theorem:** If $L/K$ is a finite Galois extension, $|\text{Gal}(L/K)| = [L : K]$.

---

## The Fundamental Theorem of Galois Theory

> **Theorem (Fundamental Theorem):** Let $L/K$ be a finite Galois extension with $G = \text{Gal}(L/K)$. There is an order-reversing bijection between:
> 1. **Intermediate fields** $E$ where $K \subseteq E \subseteq L$.
> 2. **Subgroups** $H$ of $G$ where $\{e\} \le H \le G$.
>
> The bijection is:
> - **Subgroup $\to$ Field:** $H \mapsto L^H = \{ x \in L \mid \sigma(x) = x \text{ for all } \sigma \in H \}$.
> - **Field $\to$ Subgroup:** $E \mapsto \text{Gal}(L/E)$.
>
> Inclusion reverses: $E_1 \subseteq E_2 \iff \text{Gal}(L/E_2) \le \text{Gal}(L/E_1)$.

## Worked Example: $\mathbb{Q}(\sqrt{2}, \sqrt{3})/\mathbb{Q}$

Let $L = \mathbb{Q}(\sqrt{2}, \sqrt{3})$, the splitting field of $(x^2-2)(x^2-3)$. $[L:\mathbb{Q}] = 4$.

The Galois group $G = \text{Gal}(L/\mathbb{Q})$ has order 4:

| Element | $\sqrt{2} \mapsto$ | $\sqrt{3} \mapsto$ | Order |
|---------|-------------------|-------------------|-------|
| $e$ | $\sqrt{2}$ | $\sqrt{3}$ | 1 |
| $\sigma$ | $-\sqrt{2}$ | $\sqrt{3}$ | 2 |
| $\tau$ | $\sqrt{2}$ | $-\sqrt{3}$ | 2 |
| $\sigma\tau$ | $-\sqrt{2}$ | $-\sqrt{3}$ | 2 |

This is the Klein four-group $V_4$.

### Fixed Fields

- $L^{\{e,\sigma\}}$: $\sigma$ fixes $\sqrt{3}$, negates $\sqrt{2}$. Fixed: $a + c\sqrt{3} = \mathbb{Q}(\sqrt{3})$.
- $L^{\{e,\tau\}}$: $\tau$ fixes $\sqrt{2}$. Fixed: $\mathbb{Q}(\sqrt{2})$.
- $L^{\{e,\sigma\tau\}}$: $\sigma\tau$ fixes $\sqrt{6}$. Fixed: $\mathbb{Q}(\sqrt{6})$.

## Challenge Problem

Let $L = \mathbb{Q}(\zeta_5)$ where $\zeta_5 = e^{2\pi i/5}$. $[L:\mathbb{Q}] = 4$. Determine $\text{Gal}(L/\mathbb{Q})$ and its subgroups.

<details>
  <summary>
    Solution Sketch
  </summary>
  <div>
    Any $\sigma$ maps $\zeta_5$ to $\zeta_5^k$ for $k \in \{1,2,3,4\}$. $\operatorname{Gal}(L/\mathbb{Q}) \cong (\mathbb{Z}/5\mathbb{Z})^\times \cong \mathbb{Z}/4\mathbb{Z}$ is cyclic. The only non-trivial subgroup is $\{0,2\}$, whose fixed field is $\mathbb{Q}(\sqrt{5})$.
  </div>
</details>
