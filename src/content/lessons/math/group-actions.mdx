# Group Actions and Orbit-Stabilizer

How do we study symmetries of structures that are not themselves groups? For example, when a group of rotations acts on the vertices of a square, or when we color the faces of a cube, we are seeing a group interact with a set of objects. 

In abstract algebra, this interaction is formalized as a **group action**. Studying group actions reveals deep connections between the elements that move objects and the elements that leave them fixed.

## The Motivating Problem: Symmetries of a Square

Let $D_4 = \{R_0, R_{90}, R_{180}, R_{270}, H, V, D_1, D_2\}$ be the dihedral group of symmetries of a square, and let $X = \{1, 2, 3, 4\}$ be the set of its vertices, labeled counterclockwise.

```
  1-----2
  |     |
  |     |
  4-----3
```

Each symmetry in $D_4$ maps vertices to other vertices. For example:
- The rotation $R_{90}$ maps $1 \to 2$, $2 \to 3$, $3 \to 4$, $4 \to 1$.
- The horizontal reflection $H$ (across the horizontal midline) swaps $1 \leftrightarrow 4$ and $2 \leftrightarrow 3$.

Let's pick a single vertex, say $1$. 
- Which elements in $D_4$ leave vertex $1$ in its exact position?
- Which vertices can vertex $1$ be sent to by the symmetries in $D_4$?

## Formalization: Group Actions, Orbits, and Stabilizers

Let $G$ be a group and $X$ a set. A **group action** of $G$ on $X$ is a function $G \times X \to X$ (written as $(g, x) \mapsto g \cdot x$) satisfying two axioms:

1. **Identity:** $e \cdot x = x$ for all $x \in X$.
2. **Compatibility:** $(g \cdot h) \cdot x = g \cdot (h \cdot x)$ for all $g, h \in G$ and $x \in X$.

### Orbits

The **orbit** of an element $x \in X$ is the set of all elements in $X$ that $x$ can be mapped to by actions in $G$:
$$G \cdot x = \{ g \cdot x \mid g \in G \}$$
The orbits partition the set $X$ into disjoint equivalence classes.

### Stabilizers

The **stabilizer** of an element $x \in X$ (denoted $G_x$) is the set of all elements in $G$ that leave $x$ fixed:
$$G_x = \{ g \in G \mid g \cdot x = x \}$$
**Theorem:** For any $x \in X$, the stabilizer $G_x$ is a **subgroup** of $G$.

## The Orbit-Stabilizer Theorem

There is a fundamental relation between the size of the orbit of an element and the size of its stabilizer:

> **Theorem (Orbit-Stabilizer Theorem):** Let $G$ be a finite group acting on a set $X$. For any $x \in X$, the size of the orbit of $x$ multiplied by the size of the stabilizer of $x$ equals the order of the group $G$:
> $$|G \cdot x| \cdot |G_x| = |G|$$
> Equivalently, $|G \cdot x| = [G : G_x]$ (the index of the stabilizer subgroup).

### Worked Example: Verifying Orbit-Stabilizer on a Square

Let's verify the theorem for vertex $1$ of the square under the action of $D_4$ ($|G| = 8$):
- **Orbit of 1:** We can map vertex 1 to any vertex in $\{1, 2, 3, 4\}$ using rotations. So $G \cdot 1 = \{1, 2, 3, 4\}$, which has size $4$.
- **Stabilizer of 1:** As we saw in the quiz, $G_1 = \{R_0, D_1\}$, which has size $2$.
- **Check:** $|G \cdot 1| \cdot |G_1| = 4 \cdot 2 = 8 = |D_4|$.
The theorem holds!

## Burnside's Lemma: Counting Orbits

How many distinct configurations exist up to rotational symmetry? For example, if we want to color the corners of a square using $2$ colors (Black and White), how many unique colored squares exist if we can rotate them?

This is solved by **Burnside's Lemma**:

> **Burnside's Lemma:** Let $G$ be a finite group acting on a finite set $X$. The number of orbits (which represent unique configurations up to symmetry) is:
> $$\text{Number of Orbits} = \frac{1}{|G|} \sum_{g \in G} |X^g|$$
> Where $X^g = \{ x \in X \mid g \cdot x = x \}$ is the set of elements in $X$ fixed by $g$.

## Contrast: Transitive vs Non-Transitive Actions

- **Transitive Action:** An action is transitive if there is only a single orbit. That is, for any $x, y \in X$, there exists some $g \in G$ such that $g \cdot x = y$. The action of $D_4$ on the corners of a square is transitive (any corner can be rotated to any other corner).
- **Non-Transitive Action:** There are multiple orbits. For example, if $D_4$ acts on the set of *all subsets* of corners, the orbit of a single corner $\{1\}$ is $\{\{1\}, \{2\}, \{3\}, \{4\}\}$, which is not the entire set.
