# Advanced Group Theory

Advanced group theory explores the internal structure of groups through cosets, normality, and factor groups.

## Lagrange's Theorem

**Lagrange's Theorem** states that for any finite group $G$, the order of every subgroup $H$ of $G$ divides the order of $G$.
$$|G| = [G : H] \cdot |H|$$
where $[G : H]$ is the number of distinct cosets of $H$ in $G$.

## Normal Subgroups and Quotient Groups

A subgroup $N \trianglelefteq G$ is **normal** if $gN = Ng$ for all $g \in G$. Normal subgroups allow the construction of the **quotient group** $G/N$, where elements are cosets and the operation is $(aN)(bN) = (ab)N$.

<CodeRunner 
  client:load 
  code={`def is_normal(G, H, op):
    for g in G:
        left_coset = set([op(g, h) for h in H])
        # Right coset Hg = {h * g for h in H}
        # Note: we need the inverse for a more robust check usually, 
        # but for finite groups equality of sets gH == Hg suffices.
        right_coset = set([op(h, g) for h in H])
        if left_coset != right_coset:
            return False
    return True

# In Abelian groups, all subgroups are normal
print(f"Is {H} normal in Z/12Z? {is_normal(G, H, add_mod_12)}")`}

## The First Isomorphism Theorem

Let $\phi: G \to H$ be a group homomorphism. Then:
1. $\ker(\phi)$ is a normal subgroup of $G$.
2. $\text{im}(\phi)$ is a subgroup of $H$.
3. $G/\ker(\phi) \cong \text{im}(\phi)$.

## Group Actions and Orbit-Stabilizer

A group $G$ **acts** on a set $X$ if there is a map $G \times X \to X$ satisfying identity and compatibility axioms.
The **Orbit-Stabilizer Theorem** relates the size of the orbit $G \cdot x$ and the stabilizer $G_x$:
$$|G| = |G \cdot x| \cdot |G_x|$$
