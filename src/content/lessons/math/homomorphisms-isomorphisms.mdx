# Group Homomorphisms and Isomorphism Theorems

How do we compare two different groups? For instance, does the group of rotations of a square behave like integers modulo 4? To answer this, we need structure-preserving maps called **homomorphisms**. When such a map is bijective, it is an **isomorphism**, meaning the two groups are structurally identical.

## The Motivating Problem: Modulo Arithmetic as a Map

Define $\phi: \mathbb{Z} \to \mathbb{Z}/4\mathbb{Z}$ by $\phi(x) = x \bmod 4$.

- Does $\phi$ preserve addition? $\phi(5+6) = \phi(11) = 3$ and $\phi(5) + \phi(6) = 1 + 2 = 3$. Yes.
- What elements map to the identity $0$ in $\mathbb{Z}/4\mathbb{Z}$?

## Formalization: Homomorphisms

Let $(G, \cdot)$ and $(H, *)$ be groups. $\phi: G \to H$ is a **group homomorphism** if for all $x, y \in G$:
$$\phi(x \cdot y) = \phi(x) * \phi(y)$$

If $\phi$ is bijective, it is an **isomorphism**, and we write $G \cong H$.

### Kernel and Image

- $\ker(\phi) = \{ x \in G \mid \phi(x) = e_H \}$ is a **normal subgroup** of $G$.
- $\text{Im}(\phi) = \{ \phi(x) \mid x \in G \}$ is a subgroup of $H$.

> **Theorem (Properties):** For any homomorphism $\phi: G \to H$:
> 1. $\phi(e_G) = e_H$.
> 2. $\phi(x^{-1}) = \phi(x)^{-1}$.
> 3. $\ker(\phi) \trianglelefteq G$.
> 4. $\text{Im}(\phi) \le H$.

### Proof that the Kernel is Normal

If $x \in \ker(\phi)$ then $\phi(x) = e_H$. For any $g \in G$:
$$\phi(g x g^{-1}) = \phi(g) \phi(x) \phi(g^{-1}) = \phi(g) e_H \phi(g)^{-1} = e_H$$
Hence $g x g^{-1} \in \ker(\phi)$, so $\ker(\phi) \trianglelefteq G$.

## The First Isomorphism Theorem

> **Theorem (1st Isomorphism Theorem):** If $\phi: G \to H$ is a homomorphism, then
> $$G / \ker(\phi) \cong \text{Im}(\phi)$$

```
      G
     / \
    /   \
   /     \
  /       \
 G/ker    Im(phi)
   \       /
    \     /
     \   /
      \ /
     {e_H}
```

*Proof Sketch:* Define $\psi(g \ker(\phi)) = \phi(g)$. Show $\psi$ is well-defined (independent of coset representative), a homomorphism, injective, and surjective. See the worked example for full details.

## The Second Isomorphism Theorem

> **Theorem (2nd Isomorphism Theorem):** Let $G$ be a group with $N \trianglelefteq G$ and $H \le G$. Then:
> 1. $H \cap N \trianglelefteq H$.
> 2. $HN \le G$ (and $HN = NH$).
> 3. $H / (H \cap N) \cong HN / N$.

*Proof of (1) — the rest follows from the First Isomorphism Theorem:*
Take any $x \in H \cap N$ and $h \in H$. Since $N \trianglelefteq G$, $h x h^{-1} \in N$. Since $H$ is a subgroup, $h x h^{-1} \in H$. Thus $h x h^{-1} \in H \cap N$, so $H \cap N \trianglelefteq H$.

For (3), define $\phi: H \to HN/N$ by $\phi(h) = hN$. The kernel is $H \cap N$, the image is $HN/N$, so the First Isomorphism Theorem gives $H/(H\cap N) \cong HN/N$.

## The Third Isomorphism Theorem

> **Theorem (3rd Isomorphism Theorem):** If $N \trianglelefteq G$ and $N \subseteq K \trianglelefteq G$ (so $K/N \trianglelefteq G/N$), then:
> $$(G/N) / (K/N) \cong G/K$$

*Proof:* Define $\phi: G/N \to G/K$ by $\phi(gN) = gK$. Check $\phi$ is a well-defined homomorphism with kernel $K/N$ and image $G/K$, then apply the First Isomorphism Theorem.

---

## Worked Example: $G/Z(G) \cong \text{Inn}(G)$

The **inner automorphism group** $\text{Inn}(G)$ consists of conjugation maps $\gamma_g: x \mapsto g x g^{-1}$.

Define $\phi: G \to \text{Inn}(G)$ by $\phi(g) = \gamma_g$. Since $\gamma_g \circ \gamma_h = \gamma_{gh}$ (check this!), $\phi$ is a homomorphism. The kernel is:
$$\ker(\phi) = \{ g \in G \mid \gamma_g = \text{id}_G \} = \{ g \in G \mid g x g^{-1} = x \text{ for all } x \in G \} = Z(G)$$

The image is $\text{Inn}(G)$ by definition. By the First Isomorphism Theorem:
$$G / Z(G) \cong \text{Inn}(G)$$

## Contrast: Homomorphism vs Arbitrary Map

The map $\theta: \mathbb{Z} \to \mathbb{Z}$ defined by $\theta(x) = x+1$:
- $\theta(x+y) = x+y+1$
- $\theta(x) + \theta(y) = (x+1)+(y+1) = x+y+2$
$\theta$ is **not** a homomorphism. It fails both operation preservation ($\theta(0)=1\neq0$) and identity mapping.

### Challenge Problem

Prove that $\text{Inn}(G) \trianglelefteq \text{Aut}(G)$ (inner automorphisms are normal in all automorphisms). Then compute $\text{Aut}(G)/\text{Inn}(G)$, which is called the **outer automorphism group**.

<details class="my-6 rounded-lg border border-primary/20 bg-primary/[0.03] p-4 dark:border-primary/25 dark:bg-primary/[0.06]">
  <summary class="font-semibold text-sm cursor-pointer select-none outline-none text-light-text dark:text-dark-text">
    Solution Sketch
  </summary>
  <div class="mt-3 text-sm leading-relaxed text-light-muted dark:text-dark-muted">
    For any $\phi \in \operatorname{Aut}(G)$ and $\gamma_g \in \operatorname{Inn}(G)$,
    $$(\phi \circ \gamma_g \circ \phi^{-1})(x) = \phi(g\phi^{-1}(x)g^{-1}) = \phi(g)x\phi(g)^{-1} = \gamma_{\phi(g)}(x).$$
    The conjugate of an inner automorphism is inner, so $\operatorname{Inn}(G) \trianglelefteq \operatorname{Aut}(G)$.
  </div>
</details>
