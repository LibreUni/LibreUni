# Ideals and Quotient Rings

In group theory, quotient groups require normal subgroups. In ring theory, quotient rings require subsets that absorb multiplication by any ring element — these are **ideals**.

## The Motivating Problem: Coset Multiplication in Rings

Let $R$ be a ring and $I \subseteq R$ an additive subgroup. We want the cosets $a + I$ to form a ring $R/I$ with:
$$(a+I)(b+I) = ab + I$$

For this to be well-defined, if $a_2 = a_1 + x$ and $b_2 = b_1 + y$ with $x, y \in I$, we need:
$$a_2 b_2 - a_1 b_1 = a_1 y + x b_1 + xy \in I$$

Since $x,y \in I$, $xy \in I$. But $a_1 y$ and $x b_1$ must also be in $I$ for *any* $a_1, b_1 \in R$. Thus $I$ must "absorb" multiplication by any ring element.

## Formalization: Ideals

A subset $I \subseteq R$ is a **(two-sided) ideal** if:
1. $(I, +) \le (R, +)$ (additive subgroup).
2. **Absorption:** For all $x \in I$ and $r \in R$, $r x \in I$ and $x r \in I$.

If $R$ is commutative, we only need $r x \in I$.

### Quotient Rings

> **Theorem:** If $I$ is an ideal of $R$, then $R/I$ (cosets $a+I$) forms a ring under $(a+I)+(b+I) = (a+b)+I$ and $(a+I)(b+I) = ab+I$.

## Prime and Maximal Ideals

| Ideal type | Quotient property |
|-----------|------------------|
| $P$ is **prime** | $R/P$ is an integral domain |
| $M$ is **maximal** | $R/M$ is a field |

> **Theorem (Maximal ⟹ Field):** $M$ is maximal iff $R/M$ is a field.

*Proof ($\Leftarrow$):* Suppose $R/M$ is a field. If $J$ is an ideal with $M \subsetneq J \subseteq R$, pick $a \in J \setminus M$. Then $a+M$ is non-zero in $R/M$, hence a unit, so $(a+M)(b+M) = 1+M$ for some $b$. Then $ab - 1 \in M \subset J$, so $1 \in J$, thus $J = R$. Hence $M$ is maximal.

*Proof ($\Rightarrow$):* Suppose $M$ is maximal. For any $a \notin M$, the ideal generated by $M \cup \{a\}$ is all of $R$, so $1 = m + ra$ for some $m \in M$, $r \in R$. Then $(a+M)(r+M) = ar + M = 1 - m + M = 1+M$, so $a+M$ is a unit. Hence $R/M$ is a field. $\square$

### Correspondence Theorem

If $I$ is an ideal of $R$, there is a bijection between ideals of $R$ containing $I$ and ideals of $R/I$:
$$\{ J \mid I \subseteq J \subseteq R \} \longleftrightarrow \{ J/I \mid J/I \subseteq R/I \}$$

## Ideal Lattice of $\mathbb{Z}/12\mathbb{Z}$

Every ideal of $\mathbb{Z}$ is of the form $n\mathbb{Z}$. In $\mathbb{Z}/12\mathbb{Z}$, the ideals correspond to divisors of 12:

```
                 Z/12Z
               /   |   \
              /    |    \
         (2) /  (3)| (4) \
            /     |      \
        Z/6Z    Z/4Z    Z/3Z
          |       |
          |      Z/2Z
          |       |
         {0}    {0}

(2) = {0,2,4,6,8,10} — not maximal (quotient Z/2Z is a field, so (2) is maximal in Z, but in Z/12Z it's... wait)
```

Actually, in $\mathbb{Z}$: $(2) \subset (2, x)$... The ideals of $\mathbb{Z}$ are $(n)$. Let's be precise:

| $n$ | Ideal $n\mathbb{Z}$ | Prime? | Maximal? | $R/I$ |
|-----|-------------------|--------|----------|-------|
| 12 | $12\mathbb{Z}$ | No | No | $\mathbb{Z}/12\mathbb{Z}$ (not domain) |
| 6 | $6\mathbb{Z}$ | No | No | $\mathbb{Z}/6\mathbb{Z}$ (zero-divisors) |
| 4 | $4\mathbb{Z}$ | No | No | $\mathbb{Z}/4\mathbb{Z}$ (zero-divisors) |
| 3 | $3\mathbb{Z}$ | Yes | Yes | $\mathbb{F}_3$ (field) |
| 2 | $2\mathbb{Z}$ | Yes | Yes | $\mathbb{F}_2$ (field) |

## Worked Example: Ideal Absorption in $\mathbb{Z}/n\mathbb{Z}$

<CodeRunner
  client:load
  title="Ideal Absorption Verifier"
  code={`def is_ideal_in_zn(n, zlimit):
    ideal = set((k * n) % zlimit for k in range(zlimit))
    for r in range(zlimit):
        for x in ideal:
            if (r * x) % zlimit not in ideal:
                return False, f"Fails: {r}*{x}"
    return True, f"{ideal} is an ideal"

print(is_ideal_in_zn(2, 6))
print(is_ideal_in_zn(3, 12))
print(is_ideal_in_zn(4, 12))
`}

## Contrast: Prime vs Maximal

In $\mathbb{Z}$, $\{0\}$ is prime but **not** maximal: $\{0\} \subsetneq 2\mathbb{Z} \subsetneq \mathbb{Z}$. The quotient $\mathbb{Z}/\{0\} \cong \mathbb{Z}$ is an integral domain but not a field.

### Challenge Problem

Prove that if $n$ is composite, $\mathbb{Z}/n\mathbb{Z}$ contains zero-divisors (connecting to rings-domains).

<details>
  <summary>
    Solution
  </summary>
  <div>
    If n = ab with 1 &lt; a,b &lt; n, then a·b ≡ 0 (mod n) but a,b ≠ 0 (mod n). Hence a is a zero-divisor.
  </div>
