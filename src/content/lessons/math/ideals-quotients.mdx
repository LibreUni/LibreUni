# Ideals and Quotient Rings

In group theory, we could not construct a quotient group unless the subgroup was normal. Similarly, in ring theory, we cannot construct a quotient ring unless the additive subgroup satisfies a special multiplication-absorbing property. 

Subrings with this property are called **ideals**. Furthermore, the properties of these ideals determine whether the resulting quotient ring is an integral domain or a field.

## The Motivating Problem: Coset Multiplication in Rings

Suppose we have a ring $R$ and an additive subgroup $I \subseteq R$. We want to construct a quotient ring $R/I$ whose elements are the cosets $a + I$. The addition operation is inherited from group theory:
$$(a + I) + (b + I) = (a + b) + I$$

Now we want to define multiplication on these cosets:
$$(a + I)(b + I) = ab + I$$

For this multiplication to be well-defined, the result must not depend on the choice of representatives. Suppose $a_1 + I = a_2 + I$ and $b_1 + I = b_2 + I$. This means:
$$a_2 = a_1 + x \quad \text{and} \quad b_2 = b_1 + y \quad \text{for some } x, y \in I$$

Let's multiply the representatives $a_2$ and $b_2$:
$$a_2 b_2 = (a_1 + x)(b_1 + y) = a_1 b_1 + a_1 y + x b_1 + xy$$

For $(a_2 b_2) + I$ to equal $(a_1 b_1) + I$, the difference $(a_2 b_2 - a_1 b_1)$ must lie in $I$. This difference is:
$$a_1 y + x b_1 + xy$$

Since $x, y \in I$, and we want $I$ to be closed under addition, the term $xy$ is in $I$. But what about $a_1 y$ and $x b_1$, where $a_1, b_1$ are arbitrary elements of the ring $R$? 

To guarantee that these terms lie in $I$, our subgroup $I$ must satisfy a powerful condition: it must "absorb" multiplication by any element in the ring.

### Try It Yourself: Ideal Absorption Check

Let's test this absorbing property. Consider the ring of integers $\mathbb{Z}$ and the subset of multiples of 3:
$$3\mathbb{Z} = \{ \dots, -6, -3, 0, 3, 6, \dots \}$$

If we take any element $x \in 3\mathbb{Z}$ (say $x = 6$) and multiply it by any arbitrary integer $r \in \mathbb{Z}$ (say $r = 7$):
$$r \cdot x = 7 \cdot 6 = 42$$
Notice that $42$ is still a multiple of 3, so it remains in $3\mathbb{Z}$.

## Formalization: Ideals and Quotient Rings

Let $R$ be a ring. A subset $I \subseteq R$ is a **two-sided ideal** (or simply **ideal**) of $R$ if:
1. $(I, +)$ is a subgroup of $(R, +)$.
2. For all $x \in I$ and all $r \in R$, both $r \cdot x \in I$ and $x \cdot r \in I$.

If $R$ is commutative, we only need to check $r \cdot x \in I$.

### Quotient Rings

> **Theorem:** If $I$ is an ideal of $R$, the quotient group $R/I$ forms a ring under the operations:
> $$(a + I) + (b + I) = (a + b) + I$$
> $$(a + I)(b + I) = ab + I$$
> The multiplicative identity of $R/I$ is the coset $1 + I$.

## Prime and Maximal Ideals

In commutative rings, we classify ideals based on the properties of their quotient rings.

### Prime Ideals

An ideal $P \subsetneq R$ is a **prime ideal** if for all $a, b \in R$:
$$ab \in P \implies a \in P \quad \text{or} \quad b \in P$$
> **Theorem:** An ideal $P$ is prime if and only if the quotient ring $R/P$ is an **integral domain**.

### Maximal Ideals

An ideal $M \subsetneq R$ is a **maximal ideal** if there is no other ideal $J$ that fits strictly between $M$ and $R$. That is, if $J$ is an ideal such that $M \subseteq J \subseteq R$, then either $J = M$ or $J = R$.
> **Theorem:** An ideal $M$ is maximal if and only if the quotient ring $R/M$ is a **field**.
> *Corollary:* In a commutative ring, every maximal ideal is a prime ideal (since every field is an integral domain).

## Worked Example: Prime and Maximal Ideals in $\mathbb{Z}$

Let's analyze the ideals of the integers $\mathbb{Z}$. Every ideal of $\mathbb{Z}$ is of the form $n\mathbb{Z}$ for some integer $n$.
1. **Let $I = 6\mathbb{Z}$:**
   - Is it prime? No, because $2 \cdot 3 = 6 \in 6\mathbb{Z}$, but $2 \notin 6\mathbb{Z}$ and $3 \notin 6\mathbb{Z}$.
   - Quotient: $\mathbb{Z}/6\mathbb{Z}$ contains zero-divisors (like $2$ and $3$), so it is not an integral domain.
2. **Let $I = 5\mathbb{Z}$:**
   - Is it prime? Yes, if $ab$ is a multiple of 5, then 5 must divide $a$ or $b$ (Euclid's Lemma).
   - Is it maximal? Yes, there is no integer $n$ dividing 5 other than 1 and 5.
   - Quotient: $\mathbb{Z}/5\mathbb{Z}$ is a field.

Let's write a Python script to check if the ideal generated by a number $n$ in the modular ring $\mathbb{Z}/m\mathbb{Z}$ satisfies the ideal absorption property.

    # The subset is multiples of n modulo z_limit: I = { (k*n) % z_limit for k }
    ideal_elements = set((k * n) % z_limit for k in range(z_limit))

                return False, f"Fails absorption: {r} * {x} = {product} is not in {ideal_elements}"

## Contrast: Prime vs Maximal Ideal

In the ring of integers $\mathbb{Z}$, the trivial ideal $\{0\}$ is prime because $ab \in \{0\} \implies a = 0$ or $b = 0$. However, $\{0\}$ is **NOT** maximal, because we can fit other ideals (like $2\mathbb{Z}$) between $\{0\}$ and $\mathbb{Z}$:
$$\{0\} \subsetneq 2\mathbb{Z} \subsetneq \mathbb{Z}$$
Quotient check: $\mathbb{Z}/\{0\} \cong \mathbb{Z}$ is an integral domain (since $\{0\}$ is prime) but not a field (since $\{0\}$ is not maximal).
