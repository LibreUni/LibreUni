# Integration: The Riemann Integral

Integration is traditionally introduced as finding the "area under a curve." Historically, Archimedes used the method of exhaustion, but a modern mathematical treatment requires the formal definition of partitions and Riemann (or Darboux) sums to define integrability.

---

## Partitions and Darboux Sums

Let $[a, b]$ be a closed and bounded interval.

<MathStatement id="partition-definition" kind="definition" title="Partition and Darboux Sums" number="1">

A **partition** $P$ of $[a, b]$ is a finite set of points $\{x_0, x_1, \dots, x_n\}$ such that:

$$a = x_0 < x_1 < x_2 < \dots < x_n = b.$$

For each subinterval $[x_{i-1}, x_i]$, we define:

$$\Delta x_i = x_i - x_{i-1}.$$

Let $f: [a, b] \to \mathbb{R}$ be a bounded function. For each subinterval, let:

$$m_i = \inf \{f(x) \mid x \in [x_{i-1}, x_i]\} \quad \text{and} \quad M_i = \sup \{f(x) \mid x \in [x_{i-1}, x_i]\}.$$

We define the **Lower Darboux Sum** $L(f, P)$ and **Upper Darboux Sum** $U(f, P)$ as:

$$L(f, P) = \sum_{i=1}^n m_i \Delta x_i \quad \text{and} \quad U(f, P) = \sum_{i=1}^n M_i \Delta x_i.$$

</MathStatement>

Clearly, for any partition $P$, $L(f, P) \le U(f, P)$. If we refine a partition (add more points), the lower sum increases (or stays same) and the upper sum decreases (or stays same).

## Riemann Integrability

We define the actual integral by taking supremum and infimum over all possible partitions.

<MathStatement id="riemann-integrable" kind="definition" title="Riemann Integrability" number="2">

Let $\mathcal{P}$ be the set of all possible partitions of $[a, b]$. We define the **lower integral** and **upper integral** of $f$ on $[a, b]$ by:

$$\underline{\int_a^b} f(x) \, dx = \sup \{L(f, P) \mid P \in \mathcal{P}\},$$
$$\overline{\int_a^b} f(x) \, dx = \inf \{U(f, P) \mid P \in \mathcal{P}\}.$$

If the lower and upper integrals are equal, we say that $f$ is **Riemann integrable** on $[a, b]$. The common value is denoted by:

$$\int_a^b f(x) \, dx.$$

</MathStatement>

Let us write a Python script to compute the upper and lower Darboux sums for $f(x) = x^2$ on $[0, 2]$ using uniform partitions of increasing size. This shows how they converge to the true value of $8/3 \approx 2.6667$.

<CodeRunner client:load
  code={`import numpy as np

def f(x): return x**2

a, b = 0.0, 2.0
n_values = [4, 10, 100, 1000]

print(f"{'n':<6} | {'Lower Sum L(f, P)':<20} | {'Upper Sum U(f, P)':<20}")
print("-" * 52)

for n in n_values:
    x_coords = np.linspace(a, b, n + 1)
    lower_sum = 0.0
    upper_sum = 0.0

    for i in range(n):
        x_left = x_coords[i]
        x_right = x_coords[i+1]
        dx = x_right - x_left

        # Since f(x)=x^2 is strictly increasing on [0, 2]
        # inf is at the left endpoint, sup is at the right endpoint
        mi = f(x_left)
        Mi = f(x_right)

        lower_sum += mi * dx
        upper_sum += Mi * dx

    print(f"{n:<6} | {lower_sum:<20.6f} | {upper_sum:<20.6f}")
`}

## Integrability of Continuous Functions

Not all bounded functions are integrable (e.g., the Dirichlet function is not). However, continuity is a sufficient condition for integrability.

<MathStatement id="continuous-integrable" kind="theorem" title="Integrability of Continuous Functions" number="1">

If $f: [a, b] \to \mathbb{R}$ is continuous on $[a, b]$, then $f$ is Riemann integrable on $[a, b]$.

</MathStatement>

*Proof sketch:* Since $[a, b]$ is closed and bounded, any continuous function $f$ on it is *uniformly continuous*. This allows us to make the difference $M_i - m_i$ arbitrarily small for a sufficiently fine partition, which forces $U(f, P) - L(f, P) < \epsilon$.

## Mean Value Theorem for Integrals

This theorem guarantees that a continuous function must attain its average value.

<MathStatement id="mvt-integrals" kind="theorem" title="Mean Value Theorem for Integrals" number="2" proofId="mvt-integrals-proof">

If $f: [a, b] \to \mathbb{R}$ is continuous, then there exists some $c \in [a, b]$ such that:

$$\int_a^b f(x) \, dx = f(c)(b - a).$$

</MathStatement>

<MathProof id="mvt-integrals-proof" for="mvt-integrals">

Since $f$ is continuous on $[a, b]$, by the Extreme Value Theorem (EVT), it attains a minimum value $m$ and a maximum value $M$ on $[a, b]$. Thus, for all $x \in [a, b]$:

$$m \le f(x) \le M.$$

Using the monotonicity property of the Riemann integral:

$$\int_a^b m \, dx \le \int_a^b f(x) \, dx \le \int_a^b M \, dx \implies m(b - a) \le \int_a^b f(x) \, dx \le M(b - a).$$

Dividing by the interval length $(b - a) > 0$:

$$m \le \frac{1}{b - a} \int_a^b f(x) \, dx \le M.$$

Let $k = \frac{1}{b - a} \int_a^b f(x) \, dx$. Since $k$ is a number between the minimum $m$ and maximum $M$ of the continuous function $f$, by the Intermediate Value Theorem (IVT), there must exist some $c \in [a, b]$ such that $f(c) = k$. Therefore,

$$f(c) = \frac{1}{b - a} \int_a^b f(x) \, dx \implies \int_a^b f(x) \, dx = f(c)(b - a).$$

</MathProof>
