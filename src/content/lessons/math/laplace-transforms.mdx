# Laplace Transforms

The Laplace transform is a powerful integral transform that maps a function from the time domain ($t$) to the complex frequency domain ($s$). In doing so, linear differential equations with initial conditions are transformed into algebraic equations, which are far easier to solve.

---

## Definition and Properties

Let $f(t)$ be a function defined for $t \ge 0$.

<MathStatement id="laplace-def" kind="definition" title="The Laplace Transform" number="1">

The **Laplace transform** of $f(t)$ is the function $F(s) = \mathcal{L}\{f(t)\}$ defined by the improper integral:

$$F(s) = \mathcal{L}\{f(t)\} = \int_0^\infty e^{-st} f(t) \, dt,$$

for all values of the complex parameter $s$ for which the integral converges.

</MathStatement>

For the transform to exist, $f(t)$ must be piecewise continuous and of **exponential order** (meaning $|f(t)| \le M e^{ct}$ for constants $M, c$).

---

## Essential Theorems

We establish the algebraic rules that make the Laplace transform valuable for solving differential equations.

<MathStatement id="laplace-theorems" kind="theorem" title="Laplace Transform Theorems" number="1">

Let $F(s) = \mathcal{L}\{f(t)\}$ and $G(s) = \mathcal{L}\{g(t)\}$.
1. **Linearity**: $\mathcal{L}\{a f(t) + b g(t)\} = a F(s) + b G(s)$.
2. **First Shifting Theorem**: $\mathcal{L}\{e^{at} f(t)\} = F(s - a)$.
3. **Transform of Derivatives**: If $f$ is continuously differentiable:
$$\mathcal{L}\{f'(t)\} = s F(s) - f(0),$$
$$\mathcal{L}\{f''(t)\} = s^2 F(s) - s f(0) - f'(0).$$
4. **Convolution Theorem**: Define the convolution $(f * g)(t) = \int_0^t f(\tau)g(t-\tau) \, d\tau$. Then:
$$\mathcal{L}\{(f * g)(t)\} = F(s) \cdot G(s).$$

</MathStatement>

## Discontinuous Forcing Functions

Laplace transforms excel at handling systems with sudden changes or shocks, which are difficult to model with classical calculus.

- **Heaviside Step Function $\mathcal{U}(t - a)$**:
$$\mathcal{U}(t - a) = \begin{cases} 0 & t < a \\ 1 & t \ge a \end{cases} \quad \implies \quad \mathcal{L}\{\mathcal{U}(t - a)\} = \frac{e^{-as}}{s}.$$
- **Dirac Delta Function $\delta(t - a)$ (Impulse)**:
$$\delta(t - a) = 0 \quad \text{for } t \neq a, \quad \int_0^\infty \delta(t-a) \, dt = 1 \quad \implies \quad \mathcal{L}\{\delta(t - a)\} = e^{-as}.$$

---

## Worked Example: Solving a Second-Order Initial Value Problem

Let us solve the initial value problem representing a mass-spring system subjected to a sudden force:

$$y'' + 4y = \delta(t - 1), \quad y(0) = 0, \quad y'(0) = 0.$$

Applying the Laplace transform to both sides:

$$\mathcal{L}\{y''\} + 4\mathcal{L}\{y\} = \mathcal{L}\{\delta(t - 1)\}$$
$$\left( s^2 Y(s) - s y(0) - y'(0) \right) + 4Y(s) = e^{-s}.$$

Substituting initial conditions $y(0) = y'(0) = 0$:

$$(s^2 + 4)Y(s) = e^{-s} \implies Y(s) = \frac{e^{-s}}{s^2 + 4}.$$

Using the fact that $\mathcal{L}^{-1}\left\{\frac{2}{s^2+4}\right\} = \sin(2t)$ and applying the second shifting theorem:

$$y(t) = \mathcal{L}^{-1}\left\{ e^{-s} \frac{1}{s^2 + 4} \right\} = \frac{1}{2} \sin(2(t - 1)) \mathcal{U}(t - 1).$$

Let us write a Python script using SymPy to calculate the Laplace transform of $e^{2t} \sin(3t)$ and the inverse Laplace transform of $\frac{1}{(s-1)(s-2)}$.
