# Limits and Continuity

Calculus is built on the concept of **change over zero**. To make sense of "instantaneous" change, we need a way to talk about what a function does as it gets closer and closer to a point, even if it never actually reaches it. This is formalised through the limit.

---

## The Formal Definition of a Limit

Intuitively, the limit of $f(x)$ as $x$ approaches $c$ is $L$ if we can make $f(x)$ arbitrarily close to $L$ by making $x$ sufficiently close to $c$. This was formalised by Weierstrass using the $\epsilon$-$\delta$ definition.

<MathStatement id="limit-definition" kind="definition" title="Limit of a Function" number="1">

Let $f: D \to \mathbb{R}$ be a function defined on a subset $D \subseteq \mathbb{R}$, and let $c$ be a limit point of $D$. We say that

$$\lim_{x \to c} f(x) = L$$

if for every $\epsilon > 0$, there exists a $\delta > 0$ such that for all $x \in D$:

$$0 < |x - c| < \delta \implies |f(x) - L| < \epsilon.$$

</MathStatement>

We can visualize this definition as a game: if someone gives us a tolerance $\epsilon$ (the target zone in the $y$-axis), we must find a window $\delta$ (the safe zone in the $x$-axis) such that all outputs of $f(x)$ fall within the target zone.

For the linear example, the horizontal $\epsilon$-band pulls back to a vertical $\delta$-band. The diagram shows why choosing $\delta=\epsilon/3$ works:

For $f(x) = 3x - 1$, let us find $\delta$ for a given $\epsilon$ at $c = 2$, where $L = 5$. We want $|(3x - 1) - 5| < \epsilon$, which simplifies to $|3x - 6| < \epsilon \iff 3|x - 2| < \epsilon \iff |x - 2| < \epsilon / 3$. Thus, choosing $\delta = \epsilon / 3$ satisfies the condition.

## Fundamental Limit Theorems

Using the $\epsilon$-$\delta$ definition, we can prove the algebraic properties of limits and bounds.

<MathStatement id="squeeze-theorem" kind="theorem" title="Squeeze Theorem" number="1" proofId="squeeze-theorem-proof">

Let $I$ be an open interval containing $c$, except possibly at $c$ itself. Let $f, g, h$ be functions defined on $I$. If for all $x \in I \setminus \{c\}$,

$$g(x) \le f(x) \le h(x)$$

and $\lim_{x \to c} g(x) = \lim_{x \to c} h(x) = L$, then:

$$\lim_{x \to c} f(x) = L.$$

</MathStatement>

<MathProof id="squeeze-theorem-proof" for="squeeze-theorem">

Let $\epsilon > 0$ be given.
1. Since $\lim_{x \to c} g(x) = L$, there exists $\delta_1 > 0$ such that $0 < |x - c| < \delta_1 \implies |g(x) - L| < \epsilon \iff L - \epsilon < g(x) < L + \epsilon$.
2. Since $\lim_{x \to c} h(x) = L$, there exists $\delta_2 > 0$ such that $0 < |x - c| < \delta_2 \implies |h(x) - L| < \epsilon \iff L - \epsilon < h(x) < L + \epsilon$.
3. Let $\delta = \min(\delta_1, \delta_2)$. For $0 < |x - c| < \delta$, both inequalities hold. Since $g(x) \le f(x) \le h(x)$, we have:
$$L - \epsilon < g(x) \le f(x) \le h(x) < L + \epsilon.$$
Thus, $L - \epsilon < f(x) < L + \epsilon \iff |f(x) - L| < \epsilon$, proving $\lim_{x \to c} f(x) = L$.

</MathProof>

A classic application of the Squeeze Theorem is proving $\lim_{x \to 0} x^2 \sin(1/x) = 0$. Since $-1 \le \sin(1/x) \le 1$, multiplying by $x^2$ yields $-x^2 \le x^2 \sin(1/x) \le x^2$. Since $\lim_{x \to 0} -x^2 = \lim_{x \to 0} x^2 = 0$, the limit of the middle function is 0.

## Continuity

Continuity formalises the idea of a function having "no gaps or jumps."

<MathStatement id="continuity-definition" kind="definition" title="Continuity" number="2">

A function $f: D \to \mathbb{R}$ is continuous at a point $c \in D$ if:

$$\lim_{x \to c} f(x) = f(c).$$

The function is continuous on an interval $I \subseteq D$ if it is continuous at every $c \in I$.

</MathStatement>

## The Intermediate Value Theorem (IVT)

Continuous functions possess topological properties that guarantee they do not skip values.

<MathStatement id="ivt" kind="theorem" title="Intermediate Value Theorem (IVT)" number="2">

If $f: [a, b] \to \mathbb{R}$ is continuous and $u$ is any number between $f(a)$ and $f(b)$, then there exists at least one $c \in (a, b)$ such that:

$$f(c) = u.$$

</MathStatement>

We can use the IVT to guarantee the existence of roots and find them numerically using the **Bisection Method**, which recursively halves the interval where a sign change is guaranteed.

<CodeRunner
  client:load
  code={`def bisection_method(f, a, b, tol=1e-5):
    if f(a) * f(b) >= 0:
        raise ValueError("IVT condition not met: f(a) and f(b) must have opposite signs.")

    while (b - a) / 2.0 > tol:
        midpoint = (a + b) / 2.0
        if f(midpoint) == 0:
            return midpoint
        elif f(a) * f(midpoint) < 0:
            b = midpoint
        else:
            a = midpoint
    return (a + b) / 2.0

# Find the root of f(x) = x^3 - x - 2 on [1, 2]
f = lambda x: x**3 - x - 2
root = bisection_method(f, 1, 2)
print(f"Calculated Root: {root:.6f}")
print(f"f(root) = {f(root):.6f}")
`}
/>

## The Extreme Value Theorem (EVT)

Continuity on closed and bounded (compact) intervals guarantees bounds.

<MathStatement id="evt" kind="theorem" title="Extreme Value Theorem (EVT)" number="3">

If $f: [a, b] \to \mathbb{R}$ is a continuous function, then $f$ must attain a maximum and a minimum at least once. That is, there exist $c, d \in [a, b]$ such that:

$$f(c) \le f(x) \le f(d) \quad \text{for all } x \in [a, b].$$

</MathStatement>
