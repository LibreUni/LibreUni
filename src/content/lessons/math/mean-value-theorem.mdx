# Mean Value Theorem

The Mean Value Theorem (MVT) is one of the most critical theoretical tools in calculus. It establishes a direct connection between the average rate of change over an interval (global behavior) and the instantaneous rate of change at a point within that interval (local behavior).

---

## Rolle's Theorem: The Foundation

Before proving the general Mean Value Theorem, we must prove its special case, Rolle's Theorem.

<MathStatement id="rolles-theorem" kind="theorem" title="Rolle's Theorem" number="1" proofId="rolles-theorem-proof">

Let $f: [a, b] \to \mathbb{R}$ be a function that satisfies:
1. $f$ is continuous on the closed interval $[a, b]$.
2. $f$ is differentiable on the open interval $(a, b)$.
3. $f(a) = f(b)$.

Then there exists at least one $c \in (a, b)$ such that $f'(c) = 0$.

</MathStatement>

<MathProof id="rolles-theorem-proof" for="rolles-theorem">

By the Extreme Value Theorem (EVT), the continuous function $f$ on the compact set $[a, b]$ must attain a maximum $M$ and a minimum $m$.
1. If $M = m$, then $f$ is a constant function on $[a, b]$. Therefore, $f'(x) = 0$ for all $x \in (a, b)$, and any $c \in (a, b)$ satisfies the theorem.
2. If $M \neq m$, then at least one of these values must be attained at an interior point $c \in (a, b)$ since $f(a) = f(b)$. Suppose the maximum is attained at $c \in (a, b)$, so $f(c) = M$.
   - Since $f$ is differentiable at $c$, we examine the limit of the difference quotient.
   - For $x > c$, $f(x) - f(c) \le 0 \implies \frac{f(x) - f(c)}{x - c} \le 0$. Thus, $f'(c) = \lim_{x \to c^+} \frac{f(x)-f(c)}{x-c} \le 0$.
   - For $x < c$, $f(x) - f(c) \le 0 \implies \frac{f(x) - f(c)}{x - c} \ge 0$. Thus, $f'(c) = \lim_{x \to c^-} \frac{f(x)-f(c)}{x-c} \ge 0$.
   - Since the left-hand and right-hand limits must match, we conclude $f'(c) = 0$. A similar proof holds if the minimum is attained at $c$.

</MathProof>

## The Mean Value Theorem

We generalize Rolle's Theorem to functions where $f(a) \neq f(b)$ by tilting the coordinate system.

<MathStatement id="mean-value-theorem" kind="theorem" title="Mean Value Theorem (MVT)" number="2" proofId="mvt-proof" dependsOn={[{ id: "rolles-theorem", label: "Rolle's Theorem" }]}>

Let $f: [a, b] \to \mathbb{R}$ be a function satisfying:
1. $f$ is continuous on $[a, b]$.
2. $f$ is differentiable on $(a, b)$.

Then there exists at least one $c \in (a, b)$ such that:

$$f'(c) = \frac{f(b) - f(a)}{b - a}.$$

</MathStatement>

The theorem identifies a point where the tangent slope equals the secant slope; the interactive exploration above lets us inspect that relationship directly.

<MathProof id="mvt-proof" for="mean-value-theorem">

We define an auxiliary function $g(x)$ representing the vertical distance between the curve $y = f(x)$ and the secant line passing through $(a, f(a))$ and $(b, f(b))$:

$$g(x) = f(x) - \left[ f(a) + \frac{f(b) - f(a)}{b - a}(x - a) \right].$$

1. Since $f$ is continuous on $[a, b]$, $g$ is continuous on $[a, b]$.
2. Since $f$ is differentiable on $(a, b)$, $g$ is differentiable on $(a, b)$.
3. Evaluate $g$ at the endpoints:
   - $g(a) = f(a) - f(a) = 0$.
   - $g(b) = f(b) - \left[ f(a) + f(b) - f(a) \right] = 0$.

Since $g(a) = g(b) = 0$, $g$ satisfies all conditions of Rolle's Theorem. Thus, there exists $c \in (a, b)$ such that $g'(c) = 0$. Differentiating $g(x)$:

$$g'(x) = f'(x) - \frac{f(b) - f(a)}{b - a}.$$

Evaluating at $x = c$:

$$0 = f'(c) - \frac{f(b) - f(a)}{b - a} \implies f'(c) = \frac{f(b) - f(a)}{b - a}.$$

</MathProof>

Let us write a Python script to verify the MVT for $f(x) = x^3 - 3x$ on $[0, 2]$. The secant slope is $(f(2)-f(0))/(2-0) = (2-0)/2 = 1.0$. We find $c \in (0, 2)$ where $f'(c) = 3c^2 - 3 = 1.0 \implies 3c^2 = 4 \implies c = \sqrt{4/3} \approx 1.1547$.

<CodeRunner client:load
  code={`import numpy as np
import matplotlib.pyplot as plt

def f(x): return x**3 - 3*x
def df(x): return 3*x**2 - 3

a, b = 0.0, 2.0

# Find c in (a, b) such that df(c) == secant_slope

plt.scatter([c], [f(c)], color='black', zorder=5)

plt.title("Mean Value Theorem Visualization")

## Critical Corollaries of the MVT

The MVT allows us to transition from local derivative properties to global function properties.

<MathStatement id="zero-derivative-constant" kind="corollary" title="Zero Derivative Implies Constant" number="1" proofId="zero-derivative-proof" dependsOn={[{ id: "mean-value-theorem", label: "MVT" }]}>

If $f'(x) = 0$ for all $x$ in an open interval $I$, then $f$ is constant on $I$.

</MathStatement>

<MathProof id="zero-derivative-proof" for="zero-derivative-constant">

Pick any two points $x_1, x_2 \in I$ with $x_1 < x_2$. Since $f$ is differentiable on $I$, it is continuous on $[x_1, x_2]$ and differentiable on $(x_1, x_2)$. By the MVT, there exists $c \in (x_1, x_2)$ such that:

$$f'(c) = \frac{f(x_2) - f(x_1)}{x_2 - x_1}.$$

Since $f'(x) = 0$ for all $x \in I$, we must have $f'(c) = 0$. Therefore,

$$\frac{f(x_2) - f(x_1)}{x_2 - x_1} = 0 \implies f(x_2) - f(x_1) = 0 \implies f(x_2) = f(x_1).$$

Since this holds for any $x_1, x_2 \in I$, $f$ is constant on $I$.

</MathProof>

## Cauchy's Generalized Mean Value Theorem

We can generalize the MVT to compare two functions parametrically.

<MathStatement id="cauchy-mvt" kind="theorem" title="Cauchy's Mean Value Theorem" number="3" proofId="cauchy-mvt-proof" dependsOn={[{ id: "rolles-theorem", label: "Rolle's Theorem" }]}>

Let $f, g: [a, b] \to \mathbb{R}$ be continuous on $[a, b]$ and differentiable on $(a, b)$. If $g'(x) \neq 0$ for all $x \in (a, b)$, then there exists some $c \in (a, b)$ such that:

$$\frac{f'(c)}{g'(c)} = \frac{f(b) - f(a)}{g(b) - g(a)}.$$

</MathStatement>

<MathProof id="cauchy-mvt-proof" for="cauchy-mvt">

First, note $g(b) - g(a) \neq 0$, because if $g(b) = g(a)$, Rolle's Theorem would imply there is a point where $g'(c) = 0$, violating the hypothesis. We define:

$$h(x) = [f(b) - f(a)]g(x) - [g(b) - g(a)]f(x).$$

1. $h$ is continuous on $[a, b]$ and differentiable on $(a, b)$.
2. Compute $h(a) = f(b)g(a) - g(b)f(a)$ and $h(b) = f(b)g(a) - g(b)f(a)$. Thus $h(a) = h(b)$.
3. By Rolle's Theorem, there exists $c \in (a, b)$ such that $h'(c) = 0$:

$$h'(c) = [f(b) - f(a)]g'(c) - [g(b) - g(a)]f'(c) = 0.$$

Dividing by $g'(c)[g(b) - g(a)]$ yields:

$$\frac{f'(c)}{g'(c)} = \frac{f(b) - f(a)}{g(b) - g(a)}.$$

</MathProof>

### L'Hôpital's Rule

Cauchy's MVT is the fundamental tool used to prove L'Hôpital's Rule for indeterminate limits of the form $0/0$ or $\infty/\infty$.

<MathStatement id="lhopital-rule" kind="theorem" title="L'Hôpital's Rule" number="4">

Let $f$ and $g$ be differentiable on an open interval containing $c$ (except possibly at $c$). If $\lim_{x \to c} f(x) = \lim_{x \to c} g(x) = 0$, and $g'(x) \neq 0$ for $x \neq c$, then:

$$\lim_{x \to c} \frac{f(x)}{g(x)} = \lim_{x \to c} \frac{f'(x)}{g'(x)},$$

provided the limit on the right exists.

</MathStatement>
