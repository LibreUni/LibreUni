# Normal Subgroups and Quotient Groups

Can we define an arithmetic on the cosets of a subgroup? Given a group $G$ and a subgroup $N$, we want to treat each coset of $N$ as a single "macro-element" and multiply them:
$$(aN) \cdot (bN) = (ab)N$$

This only works when the subgroup $N$ satisfies a special symmetry property called **normality**.

## The Problem: When Multiplication Fails

Take $D_3$ and $H = \{R_0, S_1\}$. The left cosets are:
- $C_1 = \{R_0, S_1\}$
- $C_2 = \{R_{120}, S_3\}$
- $C_3 = \{R_{240}, S_2\}$

Try multiplying $C_2 \cdot C_2$ with different representatives:
- Using $R_{120}$ and $R_{120}$: $(R_{120} H)(R_{120} H) = R_{240} H = C_3$
- Using $S_3$ and $S_3$: $(S_3 H)(S_3 H) = (S_3 \circ S_3) H = R_0 H = C_1$

Two different representatives of the same coset give different products! Coset multiplication is **not well-defined** for $H$.

Now try the **rotation subgroup** $N = \{R_0, R_{120}, R_{240}\}$. Check conjugation by a reflection $S_1$:
- $S_1 R_{120} S_1^{-1} = S_1 R_{120} S_1 = S_1 S_3 = R_{240} \in N$
- $S_1 R_{240} S_1^{-1} = S_1 R_{240} S_1 = S_1 S_2 = R_{120} \in N$

No matter what $g \in D_3$ we choose, conjugating $N$ by $g$ keeps elements inside $N$.

## Formalization: Normal Subgroups

A subgroup $N \le G$ is **normal** (denoted $N \trianglelefteq G$) if it is invariant under conjugation by any element of $G$:
$$g N g^{-1} = N \quad \text{for all } g \in G$$
where $g N g^{-1} = \{ g n g^{-1} \mid n \in N \}$.

### Equivalent Conditions

$N$ is normal in $G$ iff any of the following hold:
1. $g N g^{-1} \subseteq N$ for all $g \in G$.
2. $gN = Ng$ for all $g \in G$ (left and right cosets coincide).
3. Coset multiplication $(aN)(bN) = (ab)N$ is well-defined.

> **Theorem (Normality ⟺ Well-Defined Coset Multiplication):** Let $N \le G$. Then coset multiplication is well-defined if and only if $N \trianglelefteq G$.

*Proof:*
($\Leftarrow$) Assume $N \trianglelefteq G$. Suppose $a_1 N = a_2 N$ and $b_1 N = b_2 N$, so $a_2 = a_1 n_1$, $b_2 = b_1 n_2$ for some $n_1, n_2 \in N$. Then
$$a_2 b_2 = a_1 n_1 b_1 n_2 = a_1 (n_1 b_1) n_2$$
Since $N$ is normal, $n_1 b_1 \in N b_1 = b_1 N$, so $n_1 b_1 = b_1 n_3$ for some $n_3 \in N$. Thus
$$a_2 b_2 = a_1 b_1 n_3 n_2 = (a_1 b_1) n_4$$
where $n_4 = n_3 n_2 \in N$. Hence $(a_2 b_2) N = (a_1 b_1) N$, independent of representatives.

($\Rightarrow$) Assume coset multiplication is well-defined. For any $g \in G$ and $n \in N$, we need $g n g^{-1} \in N$. Choose $a_1 = g$, $a_2 = g n$ (so $a_1 N = a_2 N$), and $b_1 = g^{-1}$, $b_2 = g^{-1}$ (trivially equal). Well-definedness gives:
$$(a_2 b_2) N = (a_1 b_1) N \implies (g n g^{-1}) N = (g g^{-1}) N = e N = N$$
Hence $g n g^{-1} \in N$. So $N$ is normal. $\square$

### Quotient Groups

If $N \trianglelefteq G$, the cosets of $N$ form a group $G/N$ (the **quotient group** of $G$ by $N$) under:
$$(aN)(bN) = (ab)N$$
- Identity: $N = eN$
- Inverse: $(aN)^{-1} = a^{-1} N$
- Order: $|G/N| = |G|/|N|$ when $G$ is finite.

## Worked Example: Proving $Z(G) \trianglelefteq G$

The **center** of a group is $Z(G) = \{ z \in G \mid zg = gz \text{ for all } g \in G \}$. Let us prove $Z(G) \trianglelefteq G$.

Take any $z \in Z(G)$ and any $g \in G$. Compute:
$$g z g^{-1} = z g g^{-1} = z \in Z(G)$$
where we used that $g z = z g$ because $z$ is central. Since $g z g^{-1} \in Z(G)$ for all $g$, $Z(G)$ is normal.

Let's test this with Python: verify that the center of $D_4$ is normal.

## Contrast: Abelian vs Non-Abelian Normality

- **Abelian groups:** Every subgroup is normal, since $g h g^{-1} = h$ for all $g, h$.
- **Non-Abelian groups:** Subgroups need not be normal, as $H = \{R_0, S_1\} \not\trianglelefteq D_3$ shows.

Prove that if $N \trianglelefteq G$ and $H \le G$, then $N \cap H \trianglelefteq H$. (The intersection of a normal subgroup with an arbitrary subgroup is normal in the subgroup.)
