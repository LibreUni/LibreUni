# Normal Subgroups and Quotient Groups

Can we define an arithmetic on the cosets of a subgroup? If we have a group $G$ and a subgroup $N$, we might want to treat each coset of $N$ as a single "macro-element" and multiply them together:
$$(aN) \cdot (bN) = (ab)N$$

But does this multiplication make sense? If we choose different representatives for the same cosets, do we get the same product coset? The answer is: **not always**. It only works when the subgroup $N$ satisfies a special symmetry property called **normality**.

## The Problem: When Multiplication Fails

Let's look at the dihedral group $D_3 = \{R_0, R_{120}, R_{240}, S_1, S_2, S_3\}$ and the subgroup:
$$H = \{R_0, S_1\}$$

From the previous lesson, we know that the left cosets of $H$ are:
- $C_1 = \{R_0, S_1\}$ (which is $H$ itself, also represented as $S_1 H$)
- $C_2 = \{R_{120}, S_3\}$ (represented as $R_{120} H$ or $S_3 H$)
- $C_3 = \{R_{240}, S_2\}$ (represented as $R_{240} H$ or $S_2 H$)

Let's try to multiply coset $C_2$ by coset $C_2$:
- **Using representatives $R_{120}$ and $R_{120}$:**
  $$(R_{120} H) \cdot (R_{120} H) = (R_{120} \circ R_{120}) H = R_{240} H = \{R_{240}, S_2\}$$
- **Using representatives $S_3$ and $S_3$:** (since $S_3 \in C_2$, we should be able to represent $C_2$ as $S_3 H$):
  $$(S_3 H) \cdot (S_3 H) = (S_3 \circ S_3) H = R_0 H = \{R_0, S_1\}$$

Wait! Depending on which elements we picked to represent the same coset $C_2$, we got two completely different product cosets: $C_3$ or $C_1$! 
This means coset multiplication is **not well-defined** for this subgroup $H$.

### Try It Yourself: Normality Check

Why did this fail? It failed because the left and right cosets of $H$ do not match. 
If we conjugation-test the subgroup (i.e., evaluate $g h g^{-1}$ for all $g \in G, h \in H$), we find that elements leak out of $H$.

Let's try the subgroup of rotations in $D_3$:
$$N = \{R_0, R_{120}, R_{240}\}$$

If we conjugate the elements of $N$ by a reflection, say $S_1$:
- $S_1 \circ R_0 \circ S_1^{-1} = R_0 \in N$
- $S_1 \circ R_{120} \circ S_1^{-1} = S_1 \circ R_{120} \circ S_1 = S_1 \circ S_3 = R_{240} \in N$
- $S_1 \circ R_{240} \circ S_1^{-1} = S_1 \circ R_{240} \circ S_1 = S_1 \circ S_2 = R_{120} \in N$

No matter what element $g \in D_3$ we choose, conjugating $N$ by $g$ keeps all elements inside $N$.

## Formalization: Normal Subgroups

Let $G$ be a group. A subgroup $N \le G$ is called a **normal subgroup** (denoted $N \trianglelefteq G$) if it is invariant under conjugation by any element of $G$. Formally:
$$g N g^{-1} = N \quad \text{for all } g \in G$$
Where $g N g^{-1} = \{ g n g^{-1} \mid n \in N \}$.

### Equivalent Conditions for Normality

A subgroup $N$ is normal in $G$ if and only if any of the following hold:
1. $g N g^{-1} \subseteq N$ for all $g \in G$.
2. $gN = Ng$ for all $g \in G$ (left cosets equal right cosets).
3. The product of two left cosets is well-defined: $(aN)(bN) = (ab)N$.

### Quotient Groups

If $N \trianglelefteq G$, the set of all cosets of $N$ in $G$ forms a group under the operation:
$$(aN)(bN) = (ab)N$$
This group is called the **quotient group** (or factor group) of $G$ by $N$, denoted **$G/N$**.
- **Identity element of $G/N$:** The coset $eN = N$.
- **Inverse in $G/N$:** $(aN)^{-1} = a^{-1}N$.
- **Order of $G/N$:** If $G$ is finite, then $|G/N| = |G|/|N|$.

## Worked Example: Proving Coset Multiplication is Well-Defined

Let's prove that if $N \trianglelefteq G$, then coset multiplication is well-defined.
Suppose $a_1 N = a_2 N$ and $b_1 N = b_2 N$. This means:
$$a_2 = a_1 n_1 \quad \text{and} \quad b_2 = b_1 n_2 \quad \text{for some } n_1, n_2 \in N$$
We want to show that $(a_2 b_2) N = (a_1 b_1) N$, which is equivalent to showing that $a_1^{-1} a_2 b_2 b_1^{-1} \in N$.

Let's compute:
$$a_2 b_2 = (a_1 n_1)(b_1 n_2) = a_1 (n_1 b_1) n_2$$
Since $N$ is normal, the left coset $b_1 N$ is equal to the right coset $N b_1$. Because $n_1 \in N$, the element $n_1 b_1$ lies in $N b_1 = b_1 N$. Thus, there exists some $n_3 \in N$ such that:
$$n_1 b_1 = b_1 n_3$$
Substituting this back in:
$$a_2 b_2 = a_1 (b_1 n_3) n_2 = (a_1 b_1)(n_3 n_2)$$
Since $n_3, n_2 \in N$, their product $n_3 n_2$ is some element $n_4 \in N$. So:
$$a_2 b_2 = (a_1 b_1) n_4 \implies (a_2 b_2) N = (a_1 b_1) N$$
Thus, the product does not depend on the choice of representatives!

# Rotations: R0, R90, R180, R270

# Let's represent D4 using permutations of vertices {0,1,2,3}
elements = {
    'R0': (0, 1, 2, 3), 'R90': (1, 2, 3, 0), 'R180': (2, 3, 0, 1), 'R270': (3, 0, 1, 2),

}

def compose(p1, p2):
    return tuple(p1[p2[i]] for i in range(4))

def get_inverse(g):
    perm = elements[g]
    inv = [0]*4
    for i, val in enumerate(perm):
        inv[val] = i
    return tuple(inv)

def is_subgroup_normal(subgroup):
    # Check if g * h * g^-1 is in subgroup for all g in D4 and h in subgroup
    for g in elements:
        g_perm = elements[g]
        g_inv = get_inverse(g)
        for h in subgroup:
            h_perm = elements[h]
            # g * h * g^-1
            conj = compose(compose(g_perm, h_perm), g_inv)
            # Find name of conjugated permutation
            conj_name = [name for name, perm in elements.items() if perm == conj]
            if not conj_name or conj_name[0] not in subgroup:
                return False, f"Failed: Conjugating {h} by {g} yields {conj_name[0] if conj_name else 'outside'}"
    return True, "Subgroup is normal"

# Subgroup 1: Rotations H1 = {R0, R90, R180, R270}
h1 = ['R0', 'R90', 'R180', 'R270']
print(f"Is Rotations subgroup normal? {is_subgroup_normal(h1)}")

# Subgroup 2: Horizontal reflection H2 = {R0, H}
h2 = ['R0', 'H']
print(f"Is {h2} normal? {is_subgroup_normal(h2)}")

## Contrast: Abelian vs Non-Abelian Normality

- **Abelian Groups:** In any Abelian group $G$, **every subgroup $H$ is normal**. Since $g \cdot h = h \cdot g$ for all $g \in G, h \in H$, conjugation always yields:
  $$g h g^{-1} = h g g^{-1} = h \in H$$
- **Non-Abelian Groups:** Subgroups are not normal in general. In $D_3$, the subgroup $\{R_0, S_1\}$ is not normal, whereas $\{R_0, R_{120}, R_{240}\}$ is normal.
