# Mathematical Optimization

Optimization is the science of finding the best (maximum or minimum) values of a function, possibly subject to constraints on the variables. It forms the backbone of operations research, economics, and machine learning.

---

## Unconstrained Optimization

Let $f: U \subseteq \mathbb{R}^n \to \mathbb{R}$ be a differentiable function defined on an open set $U$.

<MathStatement id="fermat-theorem" kind="theorem" title="Fermat's Theorem on Critical Points" number="1">

If $f$ has a local extremum at $\mathbf{a} \in U$, then $\mathbf{a}$ is a **critical point** of $f$, meaning:

$$\nabla f(\mathbf{a}) = \mathbf{0}.$$

</MathStatement>

To classify critical points in higher dimensions, we look at the curvature of the function using the Hessian matrix of second partial derivatives.

<MathStatement id="multivariable-second-derivative" kind="theorem" title="Second Derivative Test in R^n" number="2">

Let $f$ have continuous second partial derivatives in a neighborhood of a critical point $\mathbf{a}$ (where $\nabla f(\mathbf{a}) = \mathbf{0}$). Let $H = \nabla^2 f(\mathbf{a})$ be the $n \times n$ **Hessian matrix**:

$$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}(\mathbf{a}).$$

- If $H$ is **positive definite** (all eigenvalues are positive), then $f$ has a strict local minimum at $\mathbf{a}$.
- If $H$ is **negative definite** (all eigenvalues are negative), then $f$ has a strict local maximum at $\mathbf{a}$.
- If $H$ is **indefinite** (has both positive and negative eigenvalues), then $\mathbf{a}$ is a **saddle point**.

</MathStatement>

## Constrained Optimization: Lagrange Multipliers

If we want to maximize or minimize $f(\mathbf{x})$ subject to an equality constraint $g(\mathbf{x}) = c$, the extrema occur where the level curves of $f$ are tangent to the constraint surface. Geometrically, this means their normal vectors (gradients) must be parallel.

<MathStatement id="lagrange-multipliers" kind="theorem" title="Method of Lagrange Multipliers" number="3">

Let $f, g: U \subseteq \mathbb{R}^n \to \mathbb{R}$ be continuously differentiable. Let $S = \{\mathbf{x} \in U \mid g(\mathbf{x}) = c\}$. If $f|_S$ has an extremum at $\mathbf{a} \in S$ and $\nabla g(\mathbf{a}) \neq \mathbf{0}$, then there exists a scalar $\lambda$ (called the **Lagrange multiplier**) such that:

$$\nabla f(\mathbf{a}) = \lambda \nabla g(\mathbf{a}).$$

We solve this by finding the critical points of the auxiliary **Lagrangian function**:

$$\mathcal{L}(\mathbf{x}, \lambda) = f(\mathbf{x}) - \lambda (g(\mathbf{x}) - c).$$

</MathStatement>

## Inequality Constraints: KKT Conditions

When constraints involve inequalities (e.g., $g(\mathbf{x}) \le c$), we generalize Lagrange multipliers to the Karush-Kuhn-Tucker (KKT) conditions.

<MathStatement id="kkt-conditions" kind="theorem" title="Karush-Kuhn-Tucker (KKT) Conditions" number="4">

Let $f: \mathbb{R}^n \to \mathbb{R}$ be the objective function, and let $g_i(\mathbf{x}) \le 0$ for $i=1, \dots, m$ be inequality constraints. If $\mathbf{a}$ is a local minimum, then there exist multipliers $\mu_1, \dots, \mu_m$ such that:
1. **Stationarity**: $\nabla f(\mathbf{a}) + \sum_{i=1}^m \mu_i \nabla g_i(\mathbf{a}) = \mathbf{0}$.
2. **Primal Feasibility**: $g_i(\mathbf{a}) \le 0$ for all $i$.
3. **Dual Feasibility**: $\mu_i \ge 0$ for all $i$.
4. **Complementary Slackness**: $\mu_i g_i(\mathbf{a}) = 0$ for all $i$.

</MathStatement>

Complementary slackness implies that either the constraint is active ($g_i(\mathbf{a}) = 0$, so $\mu_i \ge 0$), or it is inactive ($g_i(\mathbf{a}) < 0$, which forces $\mu_i = 0$, meaning the constraint has no effect on the local minimum).

## Convex Optimization

If $f$ is a convex function and the feasible set is a convex set, then any local minimum is guaranteed to be a **global minimum**. This property is highly desirable in machine learning (e.g., in Support Vector Machines and linear regression).

Let us write a Python script using `scipy.optimize.minimize` to solve a constrained optimization problem: minimize $f(x, y) = (x - 1)^2 + (y - 2)^2$ subject to the inequality constraint $x + y \le 2$.

<CodeRunner client:load
  code={`import numpy as np
from scipy.optimize import minimize

def objective(v):
    x, y = v
    return (x - 1.0)**2 + (y - 2.0)**2

# Constraint: x + y <= 2 -> 2 - x - y >= 0
def constraint_func(v):
    x, y = v
    return 2.0 - (x + y)

cons = ({'type': 'ineq', 'fun': constraint_func})
initial_guess = [0.0, 0.0]

result = minimize(objective, initial_guess, constraints=cons)

print(f"Optimal solution: x = {result.x[0]:.6f}, y = {result.x[1]:.6f}")
print(f"Objective value:  {result.fun:.6f}")
print(f"Success?          {result.success}")
`}
/>
