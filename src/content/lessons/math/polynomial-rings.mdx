# Polynomial Rings and Irreducibility

When we work with polynomials, we are working within a ring structure. If the coefficients of these polynomials come from a field (like the rational numbers $\mathbb{Q}$ or real numbers $\mathbb{R}$), the resulting polynomial ring behaves remarkably like the integers: we can perform long division, find greatest common divisors, and factor polynomials uniquely into irreducible parts.

How do we perform division on polynomials, and how do we prove that a polynomial cannot be factored any further?

## The Motivating Problem: Finding Roots and Factors

Consider the polynomial:
$$p(x) = x^2 - 2$$

Does this polynomial factor over the field of rational numbers $\mathbb{Q}$? 
If it did, it would have to factor into two linear polynomials:
$$x^2 - 2 = (x - r)(x - s) \quad \text{for some } r, s \in \mathbb{Q}$$

This would mean $r$ and $s$ are roots of $p(x)$, i.e., $r^2 = 2$. But we know that $\sqrt{2}$ is irrational, so there is no rational number whose square is $2$. Thus, $x^2 - 2$ cannot be factored over $\mathbb{Q}$. It is **irreducible** over $\mathbb{Q}$.

However, if we change our field to the real numbers $\mathbb{R}$, we can factor it:
$$x^2 - 2 = (x - \sqrt{2})(x + \sqrt{2})$$

This shows that irreducibility depends entirely on the field of coefficients we choose.

### Try It Yourself: Applying the Factor Theorem

The **Factor Theorem** states that a polynomial $p(x)$ has a linear factor $(x - a)$ if and only if $a$ is a root of the polynomial ($p(a) = 0$).

Using this, let's test if $p(x) = x^3 - 3x + 2$ has $(x - 1)$ as a factor.

## Formalization: Polynomial Rings and Division

Let $F$ be a field. The **polynomial ring** $F[x]$ is the set of all polynomials in the variable $x$ with coefficients in $F$, equipped with standard addition and multiplication of polynomials.

Because $F$ is a field, we can always divide coefficients. This allows us to prove that $F[x]$ is a Euclidean Domain:

> **Theorem (Division Algorithm in $F[x]$):** Let $f(x), g(x) \in F[x]$ with $g(x) \neq 0$. Then there exist unique polynomials $q(x), r(x) \in F[x]$ such that:
> $$f(x) = q(x)g(x) + r(x)$$
> Where the degree of $r(x)$ is strictly less than the degree of $g(x)$ ($\text{deg}(r) < \text{deg}(g)$) or $r(x) = 0$.

### Roots and the Factor Theorem

- An element $a \in F$ is a **root** of $f(x) \in F[x]$ if $f(a) = 0$.
- **Factor Theorem:** $(x - a)$ divides $f(x)$ in $F[x]$ if and only if $f(a) = 0$.
- A polynomial of degree 2 or 3 is irreducible over $F$ if and only if it has no roots in $F$. (For degree 4 or higher, a polynomial can be reducible without having roots, e.g., $(x^2 + 1)(x^2 + 1)$ has no real roots but is reducible over $\mathbb{R}$).

## Irreducibility Tests in $\mathbb{Q}[x]$

To test higher-degree polynomials for irreducibility over the rationals $\mathbb{Q}$, we use two powerful results:

### Gauss's Lemma

If a polynomial $f(x) \in \mathbb{Z}[x]$ is reducible over $\mathbb{Q}[x]$, then it is also reducible over $\mathbb{Z}[x]$. Thus, to show a polynomial is irreducible over the rationals, it suffices to show it cannot be factored into integer polynomials.

### Eisenstein's Criterion

> **Theorem (Eisenstein's Criterion):** Let $f(x) = a_n x ^ n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0$ be a polynomial with integer coefficients. If there exists a prime $p$ such that:
> 1. $p$ divides $a_0, a_1, \dots, a_{n-1}$
> 2. $p$ does not divide $a_n$
> 3. $p^2$ does not divide $a_0$
>
> Then $f(x)$ is irreducible over $\mathbb{Q}$.

## Worked Example: Applying Eisenstein's Criterion

Let's test $f(x) = x^5 - 4x + 2$ for irreducibility over $\mathbb{Q}$.
- The coefficients are: $a_5 = 1$, $a_4 = 0$, $a_3 = 0$, $a_2 = 0$, $a_1 = -4$, $a_0 = 2$.
- Let's choose the prime $p = 2$.
  1. Does $2$ divide $a_0, a_1, a_2, a_3, a_4$? Yes: $2 \mid 2$, $2 \mid -4$, and $2 \mid 0$.
  2. Does $2$ divide $a_5 = 1$? No.
  3. Does $2^2 = 4$ divide $a_0 = 2$? No.
All three conditions are satisfied, so $x^5 - 4x + 2$ is irreducible over $\mathbb{Q}$.

  client:load
  title="Polynomial Long Division Simulator"
  code={`# We represent a polynomial as a list of coefficients from lowest degree to highest
# e.g., x^3 - 3x + 2 is [2, -3, 0, 1]

def poly_degree(p):
    # Find highest index with non-zero coefficient
    for i in range(len(p) - 1, -1, -1):
        if abs(p[i]) > 1e-9:
            return i
    return -1

def poly_div(f, g):
    # Divide f by g, return q, r
    deg_f = poly_degree(f)
    deg_g = poly_degree(g)
    
    if deg_g == -1:
        raise ZeroDivisionError("Cannot divide by zero polynomial")
        
    if deg_f < deg_g:
        return [0], f
        
    q = [0] * (deg_f - deg_g + 1)
    r = list(f)
    
    while True:
        deg_r = poly_degree(r)
        if deg_r < deg_g:
            break
        # Lead coefficient division
        lead_coeff = r[deg_r] / g[deg_g]
        deg_diff = deg_r - deg_g
        q[deg_diff] = lead_coeff
        
        # Subtract lead_coeff * x^deg_diff * g from r
        for i in range(deg_g + 1):
            r[i + deg_diff] -= lead_coeff * g[i]
            
    return q, r

# Divide f(x) = 3x^3 + 2x^2 + x + 1 by g(x) = x^2 + 1
# f = [1, 1, 2, 3], g = [1, 0, 1]
q, r = poly_div([1, 1, 2, 3], [1, 0, 1])
print(f"Quotient coefficients: {q}")  # should represent 3x + 2 -> [2, 3]
print(f"Remainder coefficients: {r}") # should represent -2x - 1 -> [-1, -2]
`}

## Contrast: Polynomials over a Field vs over a Ring

- **$\mathbb{Q}[x]$ (over a Field):** Is a Euclidean Domain. We can perform division, and every ideal is principal.
- **$\mathbb{Z}[x]$ (over a Ring):** Is **not** a Euclidean Domain, and not a PID. The division algorithm fails because we cannot divide coefficients freely. For example, if we try to divide $x^2$ by $2x$, we get a fraction $1/2$ which is not in $\mathbb{Z}$.
