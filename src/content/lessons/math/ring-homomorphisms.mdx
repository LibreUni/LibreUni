# Ring Homomorphisms and Isomorphism Theorems

How do we relate different rings, and does a map that preserves addition automatically preserve multiplication? 

In group theory, we studied group homomorphisms. In ring theory, we need mappings that preserve both operations (addition and multiplication) and map the multiplicative identity $1$ to $1$. These are called **ring homomorphisms**. Just like in group theory, the First Isomorphism Theorem holds for rings, linking homomorphisms directly to ideals.

## The Motivating Problem: Projection Modulo $N$

Consider the map:
$$\phi: \mathbb{Z} \to \mathbb{Z}/3\mathbb{Z} \quad \text{by} \quad \phi(x) = x \pmod 3$$

We already know this maps preserves addition. Let's check if it preserves multiplication with $x = 5$ and $y = 4$:
- Multiply then map: $\phi(5 \cdot 4) = \phi(20) = 20 \pmod 3 = 2$.
- Map then multiply: $\phi(5) \cdot \phi(4) \pmod 3 = (2 \cdot 1) \pmod 3 = 2$.

They match! Furthermore, $\phi(1) = 1 \pmod 3 = 1$. The map preserves all structures of the ring.

What is the kernel of this map? The kernel is the set of elements mapping to the additive identity $0$:
$$\ker(\phi) = 3\mathbb{Z}$$
Notice that $3\mathbb{Z}$ is an **ideal** of $\mathbb{Z}$. Is the kernel of any ring homomorphism always an ideal?

### Try It Yourself: Checking the Identity

Before we formalize, let's look at a candidate map. Let $\phi: \mathbb{Z} \to \mathbb{Z}$ be defined by:
$$\phi(x) = 0 \quad \text{for all } x \in \mathbb{Z}$$

Does this map preserve addition and multiplication?
- $\phi(x + y) = 0 = 0 + 0 = \phi(x) + \phi(y)$ (Yes)
- $\phi(x \cdot y) = 0 = 0 \cdot 0 = \phi(x) \cdot \phi(y)$ (Yes)

But is it a ring homomorphism according to our modern definition?

## Formalization: Ring Homomorphisms

Let $R$ and $S$ be rings. A function $\phi: R \to S$ is a **ring homomorphism** if for all $x, y \in R$:

1. **Addition Preservation:** $\phi(x + y) = \phi(x) + \phi(y)$
2. **Multiplication Preservation:** $\phi(x \cdot y) = \phi(x) \cdot \phi(y)$
3. **Identity Preservation:** $\phi(1_R) = 1_S$

If $\phi$ is bijective, it is a **ring isomorphism**, denoted $R \cong S$.

### Kernel and Image

For any ring homomorphism $\phi: R \to S$:
- The **kernel** of $\phi$ is the set of elements in $R$ mapping to $0_S$:
  $$\ker(\phi) = \{ x \in R \mid \phi(x) = 0_S \}$$
- The **image** of $\phi$ is the set of all outputs in $S$:
  $$\text{Im}(\phi) = \{ \phi(x) \mid x \in R \}$$

---

## Worked Example: Proving the Kernel is an Ideal

Let's prove that if $\phi: R \to S$ is a ring homomorphism, then $I = \ker(\phi)$ is an ideal of $R$.
1. **Additive Subgroup:** Since $\phi$ is a group homomorphism from $(R, +)$ to $(S, +)$, we already know $\ker(\phi)$ is an additive subgroup.
2. **Absorption Property:** Let $x \in \ker(\phi)$ (so $\phi(x) = 0$) and let $r \in R$ be any element. Let's compute $\phi(r \cdot x)$:
   $$\phi(r \cdot x) = \phi(r) \cdot \phi(x) = \phi(r) \cdot 0_S = 0_S$$
   Since $\phi(r \cdot x) = 0_S$, we have $r \cdot x \in \ker(\phi)$.
   Similarly, $\phi(x \cdot r) = \phi(x) \cdot \phi(r) = 0_S \cdot \phi(r) = 0_S \implies x \cdot r \in \ker(\phi)$.

Thus, $\ker(\phi)$ is an ideal!

---

## The First Isomorphism Theorem for Rings

Just as in groups, we can project a ring onto its image through the kernel:

> **Theorem (First Isomorphism Theorem for Rings):** If $\phi: R \to S$ is a ring homomorphism, then:
> 1. $\ker(\phi)$ is an ideal of $R$.
> 2. $\text{Im}(\phi)$ is a subring of $S$.
> 3. The quotient ring $R/\ker(\phi)$ is isomorphic to the image:
>    $$R/\ker(\phi) \cong \text{Im}(\phi)$$

The proof follows the same bijection $\psi(r + \ker(\phi)) = \phi(r)$ as in groups, and we verify that it preserves multiplication and identities as well.

## Contrast: Group vs Ring Homomorphism

Consider the map $\psi: \mathbb{Z} \to \mathbb{Z}$ defined by $\psi(x) = 2x$.
- **Is it a group homomorphism under addition?** Yes, because $\psi(x+y) = 2(x+y) = 2x + 2y = \psi(x) + \psi(y)$.
- **Is it a ring homomorphism?** No! 
  - It fails multiplication: $\psi(x \cdot y) = 2xy$, but $\psi(x)\psi(y) = 2x \cdot 2y = 4xy$.
  - It fails identity: $\psi(1) = 2 \neq 1$.

This highlights that a ring homomorphism is a much more restrictive mapping than a group homomorphism.
