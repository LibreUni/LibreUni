# Ring Homomorphisms and Isomorphism Theorems

In group theory, homomorphisms preserve one operation. In ring theory, we need maps that preserve both addition and multiplication. These are **ring homomorphisms**, and they relate directly to ideals.

## The Motivating Problem: Projection Modulo $n$

Define $\phi: \mathbb{Z} \to \mathbb{Z}/3\mathbb{Z}$ by $\phi(x) = x \bmod 3$:
- $\phi(5 \cdot 4) = \phi(20) = 2$ and $\phi(5)\phi(4) = 2 \cdot 1 = 2$ ✓
- $\phi(1) = 1$ ✓

What is $\ker(\phi)$? $\phi(x) = 0$ iff $3 \mid x$, so $\ker(\phi) = 3\mathbb{Z}$. This is an ideal of $\mathbb{Z}$.

## Formalization: Ring Homomorphisms

$\phi: R \to S$ is a **ring homomorphism** if for all $x, y \in R$:
1. $\phi(x + y) = \phi(x) + \phi(y)$
2. $\phi(x y) = \phi(x) \phi(y)$
3. $\phi(1_R) = 1_S$

If $\phi$ is bijective, it is a **ring isomorphism** ($R \cong S$).

### Kernel and Image

- $\ker(\phi) = \{ x \in R \mid \phi(x) = 0_S \}$ — always an **ideal** of $R$.
- $\text{Im}(\phi) = \{ \phi(x) \mid x \in R \}$ — a subring of $S$.

> **Theorem (Kernel is an Ideal):** If $\phi: R \to S$ is a ring homomorphism, $\ker(\phi)$ is an ideal of $R$.

*Proof:* $\phi$ is an additive group homomorphism, so $\ker(\phi)$ is an additive subgroup. For any $r \in R$, $x \in \ker(\phi)$:
$$\phi(r x) = \phi(r) \phi(x) = \phi(r) \cdot 0 = 0 \implies r x \in \ker(\phi)$$
Similarly $x r \in \ker(\phi)$. Thus $\ker(\phi)$ absorbs multiplication. $\square$

## The Isomorphism Theorems for Rings

### First Isomorphism Theorem

> **Theorem (1st Isomorphism Theorem for Rings):** If $\phi: R \to S$ is a ring homomorphism, then $R / \ker(\phi) \cong \text{Im}(\phi)$.

*Proof:* The group-level First Isomorphism Theorem gives an additive group isomorphism $\psi(r + \ker(\phi)) = \phi(r)$. We just need to check multiplication:
$$\psi((r + I)(s + I)) = \psi(rs + I) = \phi(rs) = \phi(r)\phi(s) = \psi(r+I)\psi(s+I)$$
and $\psi(1 + I) = \phi(1) = 1_S$. So $\psi$ is a ring isomorphism. $\square$

### Second Isomorphism Theorem

> **Theorem (2nd Isomorphism Theorem for Rings):** If $I$ is an ideal of $R$ and $S$ is a subring of $R$, then:
> 1. $S + I$ is a subring of $R$ (and $S \cap I$ is an ideal of $S$).
> 2. $(S + I)/I \cong S/(S \cap I)$.

### Correspondence Theorem

If $I$ is an ideal of $R$, there is a bijection between ideals of $R$ containing $I$ and ideals of $R/I$.

## Worked Example: The Evaluation Homomorphism

For a field $F$, define $\text{ev}_a: F[x] \to F$ by $\text{ev}_a(f) = f(a)$ (evaluate at $a$).

- $\text{ev}_a(f+g) = f(a)+g(a) = \text{ev}_a(f) + \text{ev}_a(g)$
- $\text{ev}_a(fg) = f(a)g(a) = \text{ev}_a(f)\,\text{ev}_a(g)$
- $\text{ev}_a(1) = 1$

The kernel is $\{ f \in F[x] \mid f(a) = 0 \}$, which is the ideal generated by $(x - a)$. By the First Isomorphism Theorem: $F[x]/(x-a) \cong F$.

## Contrast: Group vs Ring Homomorphism

$\psi: \mathbb{Z} \to \mathbb{Z}$, $\psi(x) = 2x$:
- Additive group homomorphism? Yes: $\psi(x+y) = 2(x+y) = 2x+2y = \psi(x)+\psi(y)$.
- Ring homomorphism? No: $\psi(xy) = 2xy \neq 4xy = \psi(x)\psi(y)$, and $\psi(1) = 2 \neq 1$.
