# Rings and Integral Domains

Group theory analyzes systems with one binary operation. Many familiar mathematical structures — integers, real numbers, matrices — possess two operations: addition and multiplication. When we have two operations, how do they interact? And why does division fail in some systems but work in others? This leads to **rings**.

## The Motivating Problem: Cancellation

In school algebra, if $a \cdot b = a \cdot c$, we cancel $a$ to get $b = c$, assuming $a \neq 0$. But in arithmetic modulo 6, let $a = 2$, $b = 3$, $c = 0$:
$$2 \cdot 3 \equiv 0 \equiv 2 \cdot 0 \pmod 6$$
Canceling $2$ would give $3 = 0$, which is false. **Zero-divisors** are the obstacle.

## Formalization: Rings

A **ring** $(R, +, \cdot)$ is a set with two binary operations satisfying:
1. **$(R, +)$ is an Abelian group:** addition associative, commutative, identity $0$, additive inverses $-a$.
2. **$(R, \cdot)$ is a monoid:** multiplication associative with identity $1 \neq 0$.
3. **Distributivity:** $a(b+c) = ab + ac$ and $(b+c)a = ba + ca$.

If $ab = ba$ for all $a,b$, $R$ is **commutative**.

### Units and Zero-Divisors

- $u \in R$ is a **unit** if $\exists v \in R : uv = vu = 1$. Units form a group $R^\times$.
- $a \neq 0$ is a **zero-divisor** if $\exists b \neq 0 : ab = 0$ or $ba = 0$.

### Integral Domains and Fields

- **Integral domain:** commutative ring with $1 \neq 0$ and **no zero-divisors**. Equivalently, cancellation holds: $ab = ac$, $a \neq 0 \implies b = c$.
- **Field:** commutative ring where every non-zero element is a unit.

> **Theorem:** Every field is an integral domain.

*Proof:* Suppose $F$ is a field and $a b = 0$ with $a \neq 0$. Since $a$ is a unit, multiply by $a^{-1}$: $b = a^{-1} \cdot 0 = 0$. Thus $F$ has no zero-divisors. $\square$

The converse is false: $\mathbb{Z}$ is an integral domain but not a field ($2$ has no inverse).

  Commutative Rings

## Characteristic

The **characteristic** $\text{char}(R)$ is the smallest $n > 0$ such that $n \cdot 1 = 0$. If no such $n$, $\text{char}(R) = 0$.

> **Theorem:** If $R$ is an integral domain, $\text{char}(R)$ is either $0$ or a prime $p$.

*Proof:* If $\text{char}(R) = n = ab$ with $1 < a, b < n$, then $0 = n \cdot 1 = (a \cdot 1)(b \cdot 1)$. Since $R$ has no zero-divisors, either $a \cdot 1 = 0$ or $b \cdot 1 = 0$, contradicting minimality of $n$.

---

## Worked Example: Units in $\mathbb{Z}/n\mathbb{Z}$

> **Theorem:** $a \in \mathbb{Z}/n\mathbb{Z}$ is a unit iff $\gcd(a, n) = 1$. Consequently, $\mathbb{Z}/n\mathbb{Z}$ is a field iff $n$ is prime.

*Proof of field condition:* If $n$ is prime, every $a \in \{1,\dots,n-1\}$ has $\gcd(a,n) = 1$, so all non-zero elements are units — hence $\mathbb{Z}/n\mathbb{Z}$ is a field. Conversely, if $n = ab$ with $1 < a, b < n$, then $a b \equiv 0 \pmod n$, so $a$ is a zero-divisor, not a unit. So $\mathbb{Z}/n\mathbb{Z}$ is not a field. $\square$

## Contrast: Ring, Domain, and Field

- **$\mathbb{Z}$:** Integral domain, not a field (units: $\pm 1$).
- **$\mathbb{Q}$:** Field (every non-zero element has an inverse).
- **$\mathbb{Z}/6\mathbb{Z}$:** Commutative ring, not a domain (zero-divisors 2,3,4).

### Challenge Problem

Prove: A **finite** integral domain is a field.

<details>
  <summary>
    Solution Sketch
  </summary>
  <div>
    Let R be a finite integral domain. For any non-zero a ∈ R, define f: R → R by f(x) = ax. Since R has no zero-divisors, f is injective. Since R is finite, f is also surjective, so ∃ b such that ab = 1. Hence a is a unit.
