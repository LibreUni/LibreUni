# Subgroups and Cyclic Groups

When studying a large, complex group, it is often useful to look for smaller groups contained within it. For example, within the symmetries of a triangle $D_3$, the set of rotations $\{R_0, R_{120}, R_{240}\}$ forms a group on its own under the same composition operation. 

How do we determine if a subset of a group is itself a group, and how do we construct groups using a single generator?

## The Problem: Classifying Subsets

Consider the group of integers under addition, $(\mathbb{Z}, +)$. Let's inspect three different subsets of $\mathbb{Z}$:
1. $2\mathbb{Z} = \{ \dots, -4, -2, 0, 2, 4, \dots \}$ (the even integers).
2. $2\mathbb{Z} + 1 = \{ \dots, -3, -1, 1, 3, 5, \dots \}$ (the odd integers).
3. $\mathbb{Z}_{\ge 0} = \{ 0, 1, 2, 3, \dots \}$ (the non-negative integers).

All of these are subsets of $\mathbb{Z}$. But only one of them forms a group under addition.

## Formalization: Subgroups and the Subgroup Criterion

Let $(G, \cdot)$ be a group. A subset $H \subseteq G$ is a **subgroup** of $G$ (denoted $H \le G$) if $H$ forms a group under the binary operation of $G$ restricted to $H$.

To check if a subset $H$ is a subgroup, we do not need to re-verify all group axioms (like associativity, which is inherited from $G$). Instead, we can use the **Subgroup Criterion**:

> **Theorem (Subgroup Criterion):** A subset $H$ of a group $G$ is a subgroup if and only if:
> 1. $H$ is non-empty ($e \in H$).
> 2. For all $x, y \in H$, $x \cdot y \in H$ (closure under the operation).
> 3. For all $x \in H$, $x^{-1} \in H$ (closure under inverses).

These three conditions can be combined into a single check:
$$H \le G \iff H \neq \emptyset \quad \text{and for all } x, y \in H, \ x \cdot y^{-1} \in H$$

## Cyclic Subgroups and Element Order

If we take any single element $a \in G$ and repeatedly apply the group operation to it, we generate a subgroup. Let's define the powers of $a$:
- For $n > 0$, $a^n = a \cdot a \cdot \dots \cdot a$ ($n$ times).
- $a^0 = e$.
- For $n > 0$, $a^{-n} = (a^{-1})^n$.

The set of all powers of $a$ is denoted by $\langle a \rangle$:
$$\langle a \rangle = \{ a^n \mid n \in \mathbb{Z} \}$$

It is straightforward to verify that $\langle a \rangle$ is a subgroup of $G$. We call $\langle a \rangle$ the **cyclic subgroup generated by $a$**. If there exists some $g \in G$ such that $G = \langle g \rangle$, we say $G$ is a **cyclic group**.

The **order** of an element $a \in G$ (denoted $|a|$) is the smallest positive integer $n$ such that $a^n = e$. If no such integer exists, the element has infinite order.
- If $|a| = n$, then $\langle a \rangle = \{ e, a, a^2, \dots, a^{n-1} \}$ has exactly $n$ elements.
- If $a$ has infinite order, then $\langle a \rangle$ is isomorphic to $\mathbb{Z}$.

## Worked Example: Subgroups of $\mathbb{Z}/6\mathbb{Z}$

Let's find all cyclic subgroups of the group $G = (\mathbb{Z}/6\mathbb{Z}, +)$, which represents integers modulo 6 under addition. The identity is $0$.
- $\langle 0 \rangle = \{ 0 \}$ (order 1, the trivial subgroup)
- $\langle 1 \rangle = \{ 1, 2, 3, 4, 5, 0 \} = G$ (order 6)
- $\langle 2 \rangle = \{ 2, 4, 0 \}$ (order 3)
- $\langle 3 \rangle = \{ 3, 0 \}$ (order 2)
- $\langle 4 \rangle = \{ 4, 2, 0 \}$ (order 3, same as $\langle 2 \rangle$)
- $\langle 5 \rangle = \{ 5, 4, 3, 2, 1, 0 \} = G$ (order 6, same as $\langle 1 \rangle$)

So the subgroups of $\mathbb{Z}/6\mathbb{Z}$ are $\{0\}$, $\{0, 3\}$, $\{0, 2, 4\}$, and the entire group $G$.

## Contrast: A Non-Cyclic Group

Not all groups can be generated by a single element. The simplest example is the **Klein four-group** $V_4$, which has 4 elements: $\{e, a, b, c\}$. Its operation is given by:
- $a^2 = b^2 = c^2 = e$
- $ab = ba = c$, $bc = cb = a$, $ca = ac = b$

Every non-identity element in $V_4$ has order $2$. Since there is no element of order $4$, $V_4$ cannot be generated by a single element, and thus is non-cyclic.
