# Systems of ODEs

Many real-world systems contain multiple interdependent variables (e.g., predator-prey dynamics, electrical circuits, coupled chemical reactions). These are modeled as systems of differential equations, where the evolution of each state depends on the values of the other states.

---

## Linear Systems with Constant Coefficients

A first-order coupled system of linear differential equations can be represented compactly in matrix form.

<MathStatement id="linear-system-ode" kind="definition" title="Linear ODE System" number="1">

A first-order homogeneous linear system of ODEs with constant coefficients is written as:

$$\mathbf{x}'(t) = \mathbf{A}\mathbf{x}(t)$$

where $\mathbf{x}(t) = \left( x_1(t), \dots, x_n(t) \right)^T$ is the state vector and $\mathbf{A}$ is an $n \times n$ real matrix.

</MathStatement>

We solve this system by guessing a solution of the form $\mathbf{x}(t) = e^{\lambda t} \mathbf{v}$, where $\mathbf{v}$ is a constant vector.
Differentiating this guess:

$$\mathbf{x}'(t) = \lambda e^{\lambda t} \mathbf{v}.$$

Substituting both into $\mathbf{x}' = \mathbf{A}\mathbf{x}$:

$$\lambda e^{\lambda t} \mathbf{v} = \mathbf{A}(e^{\lambda t} \mathbf{v}) \iff \mathbf{A}\mathbf{v} = \lambda \mathbf{v}.$$

Thus, $\lambda$ must be an eigenvalue of $\mathbf{A}$ and $\mathbf{v}$ must be its corresponding eigenvector!

<MathStatement id="homogeneous-solution" kind="theorem" title="General Solution of Homogeneous Systems" number="1">

Let $\mathbf{A}$ be an $n \times n$ matrix. If $\mathbf{A}$ has $n$ linearly independent eigenvectors $\mathbf{v}_1, \dots, \mathbf{v}_n$ corresponding to eigenvalues $\lambda_1, \dots, \lambda_n$, then the general solution to $\mathbf{x}' = \mathbf{A}\mathbf{x}$ is:

$$\mathbf{x}(t) = c_1 e^{\lambda_1 t} \mathbf{v}_1 + c_2 e^{\lambda_2 t} \mathbf{v}_2 + \dots + c_n e^{\lambda_n t} \mathbf{v}_n$$

where $c_1, \dots, c_n$ are arbitrary constants determined by initial conditions.

</MathStatement>

## The Matrix Exponential

For any system, even those with repeated or complex eigenvalues, the general solution can be expressed using the matrix exponential.

<MathStatement id="matrix-exponential" kind="definition" title="Matrix Exponential" number="2">

For any $n \times n$ matrix $\mathbf{A}$, the **matrix exponential** $e^{\mathbf{A} t}$ is defined by the power series:

$$e^{\mathbf{A} t} = \sum_{k=0}^\infty \frac{\mathbf{A}^k t^k}{k!} = \mathbf{I} + \mathbf{A}t + \frac{\mathbf{A}^2 t^2}{2!} + \dots$$

The unique solution to the initial value problem $\mathbf{x}' = \mathbf{A}\mathbf{x}$ with $\mathbf{x}(0) = \mathbf{x}_0$ is:

$$\mathbf{x}(t) = e^{\mathbf{A} t} \mathbf{x}_0.$$

</MathStatement>

## Phase Plane Analysis and Stability of 2D Systems

For a $2 \times 2$ linear system $\mathbf{x}' = \mathbf{A}\mathbf{x}$, the eigenvalues of $\mathbf{A}$ determine the geometry of the trajectories in the $x_1x_2$-plane (the phase plane) around the origin (equilibrium point).

Let $D = \det(\mathbf{A})$ and $T = \operatorname{tr}(\mathbf{A})$. The characteristic equation is $\lambda^2 - T\lambda + D = 0$.

<MathStatement id="phase-stability" kind="definition" title="Classification of Equilibrium Points" number="3">

The origin $\mathbf{x} = \mathbf{0}$ of a $2 \times 2$ linear system is classified as follows:
1. **Sink (Stable Node)**: Eigenvalues are real and negative ($T < 0$, $D > 0$, $T^2 - 4D \ge 0$). Trajectories spiral or flow directly into the origin.
2. **Source (Unstable Node)**: Eigenvalues are real and positive ($T > 0$, $D > 0$, $T^2 - 4D \ge 0$). Trajectories flow away from the origin.
3. **Saddle Point (Unstable)**: Eigenvalues are real and have opposite signs ($D < 0$). Trajectories approach along one eigenvector axis and depart along the other.
4. **Center (Neutrally Stable)**: Eigenvalues are purely imaginary ($T = 0$, $D > 0$). Trajectories form concentric closed loops.
5. **Spiral Sink/Source**: Eigenvalues are complex conjugates $\alpha \pm i\beta$ ($T^2 - 4D < 0$). Trajectories spiral in if $T < 0$ (sink) and spiral out if $T > 0$ (source).

</MathStatement>

Let us write a Python script to compute the eigenvalues and eigenvectors of a system, classify the origin, and simulate a few trajectories.

<CodeRunner client:load

import matplotlib.pyplot as plt

# Matrix A = [[1, -1], [2, -1]]

eigenvalues, eigenvectors = np.linalg.eig(A)
print(f"Eigenvalues: {eigenvalues}")

# Trace = 0, Det = 1 > 0 -> Pure imaginary eigenvalues +/- i -> Center
tr = np.trace(A)
det = np.linalg.det(A)

# Numerical integration of a trajectory starting at (1, 0)
def system(state, t):
    return np.dot(A, state)

from scipy.integrate import odeint
t = np.linspace(0, 10, 200)
trajectory = odeint(system, [1.0, 0.0], t)

plt.plot(trajectory[:, 0], trajectory[:, 1], label='Trajectory', color='#3b82f6')
plt.scatter([0], [0], color='red', label='Center Equilibrium')

plt.ylabel('y')

plt.grid(True)
plt.show()
