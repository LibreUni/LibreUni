# General Topology

Topology is the study of properties that are preserved under continuous deformations, such as stretching and bending, but not tearing or gluing.

## Metric Spaces

A **metric space** $(X, d)$ is a set $X$ with a distance function $d: X \times X \to \mathbb{R}$ satisfying:
1. **Positivity**: $d(x, y) \ge 0$, and $d(x, y) = 0 \iff x = y$.
2. **Symmetry**: $d(x, y) = d(y, x)$.
3. **Triangle Inequality**: $d(x, z) \le d(x, y) + d(y, z)$.

## Topological Spaces

A **topological space** $(X, \tau)$ is a set $X$ with a collection of subsets $\tau$ (the open sets) such that:
1. $\emptyset \in \tau$ and $X \in \tau$.
2. Arbitrary unions of sets in $\tau$ are in $\tau$.
3. Finite intersections of sets in $\tau$ are in $\tau$.

<CodeRunner 
  client:load 
  code={`def is_topology(X, tau):
    X_set = set(X)
    tau_sets = [set(s) for s in tau]
    
    # Empty set and X
    if set() not in tau_sets or X_set not in tau_sets:
        return False, "Missing empty set or X"
    
    # Finite Intersections
    import itertools
    for s1, s2 in itertools.combinations(tau_sets, 2):
        if s1.intersection(s2) not in tau_sets:
            return False, f"Intersection of {s1} and {s2} not in tau"

    for s1, s2 in itertools.combinations(tau_sets, 2):
        if s1.union(s2) not in tau_sets:
            return False, f"Union of {s1} and {s2} not in tau"
            
    return True, "Valid Topology"

## Continuity

A function $f: X \to Y$ is **continuous** if the preimage of every open set in $Y$ is open in $X$. 
- Formally: $f$ is continuous if $\forall V \in \tau_Y, f^{-1}(V) \in \tau_X$.

In a **Metric Space**, this is equivalent to the $\epsilon-\delta$ definition: $f$ is continuous at $x_0$ if:
$$\forall \epsilon > 0, \exists \delta > 0 \text{ such that } d_X(x, x_0) < \delta \implies d_Y(f(x), f(x_0)) < \epsilon$$

## Compactness and Connectedness

A space is **compact** if every open cover has a finite subcover. A space is **connected** if it cannot be partitioned into two disjoint non-empty open sets.
