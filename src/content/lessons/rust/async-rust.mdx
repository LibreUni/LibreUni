# Async Rust

Asynchronous programming allows a program to handle many concurrent tasks without creating a thread per task. Rust's async model is **zero-cost**: the state machines generated by `async` have minimal overhead.

{/* Source tracking: Async Rust synthesized from "The Rust Programming Language" by Klabnik & Nichols, Ch. 17 (MIT/Apache-2.0) */}

## async and await

```rust
use std::time::Duration;

async fn do_something() -> String {
    // Simulating async work
    tokio::time::sleep(Duration::from_millis(100)).await;
    String::from("done")
}

// Calling an async function returns a Future; it does nothing until awaited
async fn main() {
    let result = do_something().await;
    println!("{}", result);
}
```

## The Future Trait

An `async fn` returns a type implementing the `Future` trait:

```rust
pub trait Future {
    type Output;
    fn poll(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<Self::Output>;
}

enum Poll<T> {
    Ready(T),
    Pending,
}
```

When polled, a `Future` returns either `Ready(value)` or `Pending`. The runtime re-pollss when the future can make progress.

## Runtimes

Rust's standard library does not include a runtime. You use external runtimes, most commonly **tokio**:

```toml
[dependencies]
tokio = { version = "1", features = ["full"] }
```

```rust
#[tokio::main]
async fn main() {
    // tokio provides the executor
}
```

## Concurrent Tasks

```rust
use tokio::task;

async fn fetch_data(id: u32) -> String {
    format!("data-{}", id)
}

#[tokio::main]
async fn main() {
    let mut handles = vec![];

    for i in 0..10 {
        handles.push(task::spawn(fetch_data(i)));
    }

    for handle in handles {
        let result = handle.await.unwrap();
        println!("{}", result);
    }
}
```

`task::spawn` runs futures concurrently on a thread pool, not synchronously.

/// A minimal blocking executor
fn block_on<F: Future>(mut future: F) -> F::Output {
    let waker = std::task::Waker::noop();  // simple, not for production
    let mut cx = Context::from_waker(&waker);
    
    let mut pinned = unsafe {
        Pin::new_unchecked(&mut future)
    };
    
    loop {
        match pinned.as_mut().poll(&mut cx) {
            Poll::Ready(val) => return val,
            Poll::Pending => std::thread::sleep(Duration::from_millis(1)),
        }
    }
}

/// A simple future that yields after a delay
struct Delay {
    until: Instant,
}

impl Future for Delay {
    type Output = &'static str;

    fn poll(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<Self::Output> {
        if Instant::now() >= self.until {
            Poll::Ready("done")

            Poll::Pending
        }
