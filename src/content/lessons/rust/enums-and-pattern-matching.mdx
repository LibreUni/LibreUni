# Enums and Pattern Matching

An **enum** (enumeration) defines a type by enumerating its possible **variants**. Combined with `match`, enums provide Rust's primary mechanism for algebraic data types—a concept from functional programming languages.

{/* Source tracking: Enums and pattern matching synthesized from "The Rust Programming Language" by Klabnik & Nichols, Ch. 6 (MIT/Apache-2.0) */}

## Defining Enums

```rust
enum IpAddrKind {
    V4,
    V6,
}
```

Rust enums can **carry data**—each variant can hold values of different types and amounts:

```rust
enum IpAddr {
    V4(u8, u8, u8, u8),
    V6(String),
}

let home = IpAddr::V4(127, 0, 0, 1);
let loopback = IpAddr::V6(String::from("::1"));
```

This is more ergonomic than having separate struct types, because all variants are unified under a single type.

## The Option Enum

Rust has no `null`. Instead, it provides `Option<T>`, defined in the standard library:

```rust
enum Option<T> {
    None,
    Some(T),
}
```

```rust
let some_number = Some(5);         // Option<i32>
let absent_number: Option<i32> = None;
```

Because `Option<T>` and `T` are different types, the compiler prevents using a potentially absent value as if it were definitely present.

## The match Expression

`match` is Rust's most powerful control flow construct. It compares a value against a series of patterns and executes the matching arm:

```rust
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter,
}

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
}
```

Matches must be **exhaustive**: every possible variant must be covered. Use `_` as a catch-all pattern:

```rust
let x = 42;
match x {
    1 => println!("one"),
    2 => println!("two"),
    _ => println!("other"),  // must cover all remaining
}
```

## if let

When matching a single pattern, `if let` provides a concise alternative:

```rust
let config_max = Some(3u8);
match config_max {
    Some(max) => println!("Maximum: {}", max),
    _ => (),
}

// Equivalent, but shorter:
if let Some(max) = config_max {
    println!("Maximum: {}", max);
}
```
