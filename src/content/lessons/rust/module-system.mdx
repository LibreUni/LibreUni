# The Module System and Cargo

Rust provides a hierarchical module system for organizing code. Combined with Cargo (Rust's build tool and package manager), it enables scalable project structures.

{/* Source tracking: Modules and Cargo synthesized from "The Rust Programming Language" by Klabnik & Nichols, Ch. 7 (MIT/Apache-2.0) */}

## Crates and Packages

A **crate** is the smallest compilation unit. There are two types:
- **Binary crate**: compiles to an executable (has `main()`).
- **Library crate**: provides reusable functionality (no `main()`).

A **package** is a collection of crates defined by a `Cargo.toml` file. It can contain at most one library crate and any number of binary crates.

## Modules

Modules organize code within a crate. They can be nested and control visibility:

```rust
mod front_of_house {
    pub mod hosting {
        pub fn add_to_waitlist() {}
        fn seat_at_table() {}      // private by default
    }

    mod serving {
        fn take_order() {}
    }
}
```

Items are **private** by default (inaccessible outside their module). The `pub` keyword makes them public.

## Paths

```rust
// Absolute path
crate::front_of_house::hosting::add_to_waitlist();

// Relative path
front_of_house::hosting::add_to_waitlist();
```

The `super` keyword accesses the parent module (like `..` in filesystems):

```rust
fn serve_order() {}

mod back_of_house {
    fn fix_incorrect_order() {
        cook_order();
        super::serve_order();  // calls parent's serve_order
    }
    fn cook_order() {}
}
```

## The use Keyword

```rust
// Bring a path into scope
use crate::front_of_house::hosting;
hosting::add_to_waitlist();

// Inline path
use crate::front_of_house::hosting::add_to_waitlist;
add_to_waitlist();

// Renaming
use std::fmt::Result as FmtResult;

// Re-exporting
pub use crate::front_of_house::hosting;
```

## File System Separation

Rust maps modules to files automatically. A module declaration `mod hosting;` tells Rust to look for `hosting.rs` or `hosting/mod.rs`:

```
src/
  lib.rs              -> crate root
  front_of_house/
    mod.rs           -> front_of_house module
    hosting.rs       -> front_of_house::hosting
```
